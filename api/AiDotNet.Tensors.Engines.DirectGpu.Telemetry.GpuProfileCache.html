Remarks

This cache stores optimal GPU configurations locally, reducing network dependency and providing instant access to previously tuned configurations.

Cache Location:

  • Windows: %LOCALAPPDATA%\AiDotNet\gpu_profiles.json
  • Linux: ~/.local/share/AiDotNet/gpu_profiles.json
  • macOS: ~/Library/Application Support/AiDotNet/gpu_profiles.json

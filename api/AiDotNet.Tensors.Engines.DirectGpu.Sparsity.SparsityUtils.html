Remarks

2:4 Structured Sparsity:

A sparsity pattern where every group of 4 consecutive elements contains exactly 2 zeros. This provides predictable memory access patterns and enables hardware acceleration on AMD MI200+ and NVIDIA Ampere+ GPUs.

Benefits:

  • 2x memory compression
  • 2x compute reduction
  • Predictable memory access patterns
  • Hardware-accelerated sparse tensor cores

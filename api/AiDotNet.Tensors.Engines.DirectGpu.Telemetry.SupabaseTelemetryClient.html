Remarks

This client submits GPU A/B test results and retrieves optimal configurations from the community-shared profile database.

Privacy:

  • Machine ID is hashed (SHA256) - never sent in plain text
  • Only GPU info and performance metrics are collected
  • No personal data or file system information is transmitted
  • Opt-out is supported and respected

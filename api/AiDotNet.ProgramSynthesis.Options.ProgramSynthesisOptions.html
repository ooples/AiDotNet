Remarks

This options type is intended for use with PredictionModelBuilder to configure a code model and safe defaults (tokenization and architecture) without requiring users to construct low-level engines.

For Beginners: These settings control which built-in code model to use and how big it is. You can usually accept the defaults and only change the language and model kind.

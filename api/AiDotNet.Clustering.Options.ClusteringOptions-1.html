Remarks

This class provides common configuration options shared by most clustering algorithms. Specific clustering implementations may extend this with algorithm-specific options.

For Beginners: These are the settings you can adjust to control how the clustering algorithm works.

Common options include:

  • How many iterations to run
  • When to stop (convergence threshold)
  • Random seed for reproducibility
  • Which distance metric to use

Properties

DistanceMetric

Gets or sets the distance metric to use.

public IDistanceMetric<T>? DistanceMetric { get; set; }

Property Value

IDistanceMetric<T>

The distance metric, or null to use algorithm default (typically Euclidean).

Remarks

For Beginners: Different distance metrics are better for different types of data. Euclidean is the most common, but Cosine is better for text, and Manhattan can be better for high-dimensional data.

MaxIterations

Gets or sets the maximum number of iterations.

public int MaxIterations { get; set; }

Property Value

int

The maximum iterations. Default is 300.

Remarks

For Beginners: Clustering algorithms iterate (repeat steps) until they find a good solution or reach this limit. Higher values allow more refinement but take longer to run.

NumInitializations

Gets or sets the number of times to run with different random initializations.

public int NumInitializations { get; set; }

Property Value

int

Number of initializations. Default is 10.

Remarks

For Beginners: Since random starting points can lead to different results, running multiple times and keeping the best result helps find a better solution. Higher values improve quality but take longer.

RandomState

Gets or sets the random seed for reproducibility.

public int? RandomState { get; set; }

Property Value

int?

The random seed, or null for non-deterministic behavior.

Remarks

For Beginners: Many clustering algorithms involve randomness (like choosing initial centers). Setting this ensures you get the same results every time you run with the same data.

Tolerance

Gets or sets the convergence tolerance.

public double Tolerance { get; set; }

Property Value

double

The tolerance for declaring convergence. Default is 1e-4.

Remarks

For Beginners: When changes between iterations become smaller than this value, the algorithm stops. Smaller values mean more precise results but potentially more iterations.

Verbose

Gets or sets the verbosity level for logging.

public int Verbose { get; set; }

Property Value

int

0 for silent, 1 for progress, 2 for detailed. Default is 0.

Remarks

Design Philosophy:

This interface extends IDirectGpuBackend with capabilities for: - Multi-stream execution (compute/transfer overlap) - GPU events for inter-stream synchronization - Async memory transfers (non-blocking H2D/D2H) - Sync points for deferred synchronization

Performance Benefits:

  • Overlap compute and memory transfer operations
  • Reduce CPU blocking with async APIs
  • Enable execution graph optimization
  • Support GPU-resident tensors with lazy sync

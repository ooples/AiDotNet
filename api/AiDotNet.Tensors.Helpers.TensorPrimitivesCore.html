Remarks

This class implements Microsoft's operator pattern for hardware-accelerated operations. It automatically detects available SIMD instruction sets and dispatches to the widest available vector width for maximum performance.

SIMD Hierarchy (fastest to slowest): - AVX-512: 512-bit vectors (8 doubles or 16 floats) - Intel Skylake-X 2017+, AMD Zen 4 2022+ - AVX2: 256-bit vectors (4 doubles or 8 floats) - Intel Haswell 2013+, AMD Excavator 2015+ - SSE/NEON: 128-bit vectors (2 doubles or 4 floats) - All modern x64/ARM64 CPUs - Scalar: Fallback for unsupported hardware or remainder elements

Performance Impact: AVX-512 can process 8x more doubles (16x more floats) per instruction than scalar code, achieving 8-12x speedups for transcendental functions like Sin/Cos.

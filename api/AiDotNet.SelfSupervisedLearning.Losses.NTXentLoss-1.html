Remarks

For Beginners: NT-Xent is the loss function used in SimCLR. It encourages representations of augmented views of the same image to be similar, while pushing apart representations of different images.

How it works:

  1. For each image, create two augmented views (positive pair)
  2. Compute similarity between all pairs in the batch
  3. For each anchor, its positive pair should have highest similarity
  4. All other samples in the batch serve as negatives

Loss formula:

L = -log( exp(sim(z_i, z_j) / τ) / Σ_k exp(sim(z_i, z_k) / τ) )

Where τ (tau) is the temperature parameter.

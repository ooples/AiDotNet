Specifies the activation function to use in fused GPU operations.

public enum ActivationType

Fields

GELU = 2

Gaussian Error Linear Unit: f(x) = 0.5 * x * (1 + tanh(sqrt(2/π) * (x + 0.044715 * x³))) Popular in transformer architectures (BERT, GPT).

None = 0

No activation function applied (identity). Output = GEMM + Bias

ReLU = 1

Rectified Linear Unit: f(x) = max(0, x) Fast and effective, most common choice for hidden layers.

Sigmoid = 3

Sigmoid: f(x) = 1 / (1 + exp(-x)) Maps output to range (0, 1), useful for binary classification.

Tanh = 4

Hyperbolic Tangent: f(x) = tanh(x) = (exp(x) - exp(-x)) / (exp(x) + exp(-x)) Maps output to range (-1, 1), useful for hidden layers in some architectures.

Remarks

Purpose:

This enum provides type-safe selection of activation functions for fused GPU kernels. Using an enum instead of strings ensures compile-time validation and better IDE support.

Fused Operations:

DirectGpu kernels can fuse GEMM + Bias + Activation into a single kernel launch, eliminating memory round-trips between operations. This can provide 20-50% speedup compared to separate operations.

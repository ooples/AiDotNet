DistillationWeight

Gets or sets the weight for distillation loss relative to task loss.

public double? DistillationWeight { get; set; }

Property Value

double?

Remarks

L_total = α * L_task + (1-α) * L_distill where α = 1/(1 + DistillationWeight)

Higher values give more importance to preserving old knowledge.

Temperature

Gets or sets the distillation temperature. Higher values produce softer probability distributions.

public double? Temperature { get; set; }

Property Value

double?

Remarks

For Beginners:

  • T = 1: Normal softmax (sharp distribution)
  • T = 2-5: Soft distributions (typical for distillation)
  • T > 10: Very soft (all classes become similar)

Higher temperature helps transfer more information about uncertainty.

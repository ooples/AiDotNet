Remarks

For Beginners: Synaptic Intelligence is similar to EWC but estimates weight importance online during training rather than computing Fisher Information afterward. It tracks how much each weight contributes to loss reduction using a "path integral".

Key Insight: SI measures how much each parameter contributed to learning, not just how important it is for the current solution. This is done by integrating the gradient signal along the learning trajectory.

How it works:

  1. During training, track the running sum: ω_i += -gradient_i × Δθ_i
  2. After each task, compute importance: Ω_i = ω_i / (Δθ_i² + ξ)
  3. When learning new tasks, penalize changes: L = L_task + (c/2) × Σ Ω_i × (θ_i - θ*_i)²

The Math (Path Integral):

ω_i = Σ_t (-∂L/∂θ_i(t)) × (θ_i(t) - θ_i(t-1))

This approximates the contribution of parameter i to loss reduction.

Advantages over EWC:

  • Online computation - no need to store training data after task completion
  • Lower memory overhead than Fisher Information computation
  • Naturally handles streaming data scenarios
  • Captures dynamics of learning, not just final importance

Reference: Zenke, F., Poole, B., and Ganguli, S. "Continual Learning Through Synaptic Intelligence" (2017). ICML.

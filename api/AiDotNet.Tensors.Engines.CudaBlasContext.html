Performs SGEMM: C = alpha * A * B + beta * C

public void Sgemm(CublasOperation transa, CublasOperation transb, int m, int n, int k, float alpha, CudaDeviceMemory<float> A, int lda, CudaDeviceMemory<float> B, int ldb, float beta, CudaDeviceMemory<float> C, int ldc)

Remarks

For row-major matrices (C# default), this computes C = alpha * B^T * A^T + beta * C^T which is equivalent to C^T = alpha * A * B + beta * C.

To compute C = A * B in row-major:

  • Call Sgemm with A and B swapped, m and n swapped
  • Result is correctly in row-major format

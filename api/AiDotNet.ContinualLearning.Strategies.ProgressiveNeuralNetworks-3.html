Remarks

For Beginners: PNN prevents catastrophic forgetting by creating a new "column" (copy of the network) for each new task. Previous columns are frozen, so old knowledge is never overwritten. New columns can use lateral connections to leverage features from previous columns.

How it works:

  1. For the first task, train a standard network (column 1)
  2. For each new task, add a new column with lateral connections to previous columns
  3. Freeze all previous columns and only train the new column
  4. Lateral connections allow the new column to reuse features from old columns

The Math:

For layer l in column k:

h_k^l = f(W_k^l * h_k^(l-1) + Î£_{j<k} U_{k,j}^l * h_j^(l-1))

Where U_{k,j}^l are the lateral connection weights from column j to column k

Comparison to Other Methods:

  • EWC/MAS/SI: Add regularization to protect important parameters
  • PackNet: Prunes and freezes subnetworks for each task
  • PNN: Creates entirely new columns, completely prevents interference

PNN guarantees no forgetting but has linear memory growth with tasks. Best for scenarios with a small number of tasks where zero forgetting is critical.

Remarks

For Beginners: Barlow Twins loss encourages the cross-correlation matrix between embeddings of two augmented views to be close to the identity matrix. This avoids both collapse and redundant features.

Key insight: Instead of contrastive learning, Barlow Twins achieves representation quality through decorrelation - making different feature dimensions capture different information.

Loss components:

  • Invariance term: Diagonal elements should be 1 (same features match)
  • Redundancy reduction: Off-diagonal elements should be 0 (no redundancy)

Loss formula:

L = Σ_i (1 - C_ii)² + λ * Σ_i Σ_{j≠i} C_ij²

where C is the cross-correlation matrix and λ is the redundancy weight.

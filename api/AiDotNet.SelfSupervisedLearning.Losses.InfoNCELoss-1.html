Remarks

For Beginners: InfoNCE is the loss function used in MoCo (Momentum Contrast). It's similar to NT-Xent but designed to work efficiently with a large memory queue of negative samples.

Key differences from NT-Xent:

  • Uses a memory queue for negatives instead of in-batch negatives
  • Typically uses more negatives (65536) but smaller batch sizes
  • Asymmetric: query from online encoder, keys from momentum encoder

Loss formula:

L = -log( exp(q·k+ / τ) / (exp(q·k+ / τ) + Σ exp(q·k- / τ)) )

Remarks

For Beginners: LwF prevents forgetting by using knowledge distillation. Instead of storing old data, it stores the model's predictions (knowledge) and trains the new model to match those predictions while also learning the new task.

How it works:

  1. Before learning a new task, save the current model as "teacher"
  2. For each training sample, get predictions from both teacher and student
  3. Train to minimize: L_task (new task) + λ * L_distill (match teacher)

The Math:

L_distill = T² * KL(softmax(z_teacher/T) || softmax(z_student/T))

Where T is the temperature and z are the logits (pre-softmax outputs).

Advantages:

  • No need to store previous task data (memory efficient)
  • Works well when tasks share output space (e.g., multi-class classification)
  • Fast - no gradient projection or quadratic programming

Disadvantages:

  • Requires outputs to be comparable across tasks
  • May not work well for very different task distributions
  • Teacher model doubles memory during training

Reference: Li and Hoiem "Learning without Forgetting" (ECCV 2016)

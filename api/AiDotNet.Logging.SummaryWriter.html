Remarks

This class provides an API similar to PyTorch's torch.utils.tensorboard.SummaryWriter, making it easy to log training metrics, model weights, images, and more.

For Beginners: This is your interface to TensorBoard visualization.

During training, you use this writer to record:

  • Loss values at each step (add_scalar)
  • Model weight distributions (add_histogram)
  • Sample outputs or feature maps (add_image)
  • Model structure (add_graph)

Then you can visualize all this in TensorBoard by running: tensorboard --logdir=your_log_directory

Example usage:

using var writer = new SummaryWriter("runs/experiment_1");
for (int epoch = 0; epoch < 100; epoch++)
{
    float loss = Train();
    writer.AddScalar("loss/train", loss, epoch);
    writer.AddHistogram("layer1/weights", model.Layer1.Weights, epoch);
}

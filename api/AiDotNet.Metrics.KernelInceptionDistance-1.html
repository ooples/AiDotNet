Remarks

KID measures how similar generated images are to real images using the Maximum Mean Discrepancy (MMD) with a polynomial kernel in the feature space of an Inception network.

Advantages over FID: - Unbiased estimator (FID is biased for small sample sizes) - Provides variance estimates for the metric - Works well with smaller datasets - More robust to sample size variations

Formula: KID = MMD^2(F_real, F_generated) using polynomial kernel k(x,y) = (x^T y / d + 1)^3

Typical KID scores (multiplied by 1000 for readability): - KID < 0.5: Excellent quality - KID 0.5-2.0: Good quality - KID 2.0-5.0: Moderate quality - KID > 5.0: Poor quality

Based on "Demystifying MMD GANs" by Binkowski et al. (2018)

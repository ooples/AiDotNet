Remarks

A map-reduce chain processes collections by applying the same operation to each item (map phase), then combining all results into a single output (reduce phase).

For Beginners: Process many items the same way, then combine results.

Example - Summarizing multiple documents:

var chain = new MapReduceChain<Document, string, Summary>("DocumentSummarizer", "Summarizes multiple documents");

// Map: Process each document
chain.SetMapper(doc => SummarizeDocument(doc));

// Reduce: Combine all summaries
chain.SetReducer(summaries => CombineSummaries(summaries));

// Run on collection
var documents = new[] { doc1, doc2, doc3, doc4, doc5 };
var result = chain.Run(documents);
// Each document is summarized individually, then all summaries are combined

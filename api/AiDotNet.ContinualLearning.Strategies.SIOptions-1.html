Lambda

Gets or sets the regularization strength (c in the paper). Higher values = more protection of previous task knowledge.

public T? Lambda { get; set; }

Property Value

T

Remarks

Typical Values:

  • 0.1-1.0: Light protection, allows more plasticity
  • 1.0-10.0: Moderate protection, good balance
  • 10.0+: Strong protection, may limit learning new tasks

Default is 1.0 as suggested in the original SI paper.

Remarks

For Beginners: An ensemble approach combines the opinions of multiple "experts" (in this case, different fit detectors) to make a more reliable decision. This is similar to getting second and third opinions from different doctors before making an important medical decision.

This detector aggregates the results from multiple fit detectors, potentially giving different weights to each detector's opinion, to determine the overall fit type, confidence level, and recommendations.

Remarks

For Beginners: This constructor creates a new ensemble fit detector with a specified list of individual detectors and optional configuration settings.

You must provide at least one detector to create an ensemble. The ensemble will combine the results from all provided detectors to make its assessment.

The options parameter allows you to customize how the ensemble works, including:

  • Detector weights: How much influence each detector has on the final result
  • Maximum recommendations: The maximum number of recommendations to include in the result

Remarks

For Beginners: This method calculates a weighted average of the confidence levels reported by all individual detectors.

The process works as follows:

  1. Run each detector and get its confidence level
  2. Apply the corresponding weight to each detector's confidence level
  3. Calculate the weighted average of all confidence levels

The resulting confidence level is a value between 0 and 1, with higher values indicating greater confidence in the ensemble's assessment.

Remarks

For Beginners: This method calculates a weighted average of the fit types reported by all individual detectors, then maps this average to a final fit type.

The process works as follows:

  1. Run each detector and get its fit type assessment
  2. Apply the corresponding weight to each detector's assessment
  3. Calculate the weighted average of all assessments
  4. Map this average to a final fit type category

The mapping from average to fit type uses these thresholds (lower enum values indicate better fit):

  • <= 0.5: Good Fit (GoodFit=0)
  • <= 2.5: Moderate Fit
  • <= 5.5: Poor Fit
  • > 5.5: Very Poor Fit

Remarks

For Beginners: This method collects recommendations from all individual detectors, removes duplicates, adds a general recommendation based on the ensemble's fit type assessment, and limits the total number of recommendations if necessary.

The process works as follows:

  1. Run each detector and collect its recommendations
  2. Combine all recommendations into a single set (removing duplicates)
  3. Add a general recommendation based on the ensemble's fit type assessment
  4. Limit the number of recommendations to the maximum specified in the options

The general recommendation provides an overall assessment and suggestion based on the ensemble's fit type determination.

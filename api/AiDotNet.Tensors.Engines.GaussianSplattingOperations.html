Computes the backward pass for Gaussian covariance computation. Implements full analytical gradients for both rotation quaternions and scale vectors.

Remarks

The covariance matrix Σ is computed as Σ = R * S² * R^T where R is the rotation matrix derived from the quaternion and S is a diagonal scale matrix. This method computes the gradients dL/dq and dL/ds given dL/dΣ using the chain rule.

ProjectGaussians3DTo2D<T>(Tensor<T>, Tensor<T>, Matrix<T>, Matrix<T>, int, int, out Tensor<T>, out Tensor<T>, out Tensor<T>, out Tensor<bool>)

Projects 3D Gaussians to 2D screen space for rasterization.

public static void ProjectGaussians3DTo2D<T>(Tensor<T> means3D, Tensor<T> covariances3D, Matrix<T> viewMatrix, Matrix<T> projMatrix, int imageWidth, int imageHeight, out Tensor<T> means2D, out Tensor<T> covariances2D, out Tensor<T> depths, out Tensor<bool> visible)

Parameters

means3D Tensor<T>

3D Gaussian center positions in world space, shape [N, 3].

covariances3D Tensor<T>

3D covariance matrices in CAMERA SPACE (must be pre-transformed by view matrix rotation). Shape [N, 6] for upper-triangular storage (c00, c01, c02, c11, c12, c22) or [N, 3, 3] for full matrix.

viewMatrix Matrix<T>

4x4 view matrix to transform world coordinates to camera space.

projMatrix Matrix<T>

4x4 projection matrix containing focal lengths (p00=fx, p11=fy).

imageWidth int

Output image width in pixels.

imageHeight int

Output image height in pixels.

means2D Tensor<T>

Output 2D screen positions, shape [N, 2].

covariances2D Tensor<T>

Output 2D covariances (a, b, c for ax² + 2bxy + cy²), shape [N, 3].

depths Tensor<T>

Output depth values in camera space, shape [N].

visible Tensor<bool>

Output visibility flags, shape [N].

Type Parameters

T

The numeric type for tensor elements.

Remarks

The 2D covariance is computed using the Jacobian of perspective projection: Σ_2D = J * Σ_3D * J^T. This is the standard approach used in 3D Gaussian Splatting (Kerbl et al., 2023).

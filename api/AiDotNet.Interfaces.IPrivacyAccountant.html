Tracks cumulative privacy loss across federated learning rounds.

public interface IPrivacyAccountant

Remarks

For Beginners: Differential privacy has a finite "budget" (epsilon, delta). Each training round spends some of that budget. A privacy accountant keeps track of what was spent so you can report guarantees and enforce limits.

Remarks

For Beginners: Submodular functions have a special property called "diminishing returns" - adding more similar samples provides less and less benefit. This naturally encourages selecting diverse, representative samples.

How It Works:

  1. Start with an empty selection set
  2. Greedily add samples that maximize marginal gain
  3. The submodular property ensures diverse selections

Facility Location Objective:

F(S) = Σ max_{s∈S} sim(x, s) for all x in the data

This objective ensures selected samples cover the entire data space well.

Advantages:

  • Theoretical guarantees: greedy achieves (1-1/e) optimal
  • Naturally balances diversity and coverage
  • Computationally efficient with lazy evaluation

Reference: Wei et al. "Submodularity in Data Subset Selection and Active Learning" (ICML 2015)

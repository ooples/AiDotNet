Remarks

This optimizer mimics natural evolution: prompts are mutated, crossed over, and selected based on fitness (performance scores).

For Beginners: Evolves prompts like nature evolves species.

Example:

var optimizer = new GeneticOptimizer<double>(
    populationSize: 20,
    mutationRate: 0.1
);

var optimized = optimizer.Optimize(
    initialPrompt: "Classify sentiment:",
    evaluationFunction: prompt => EvaluateAccuracy(prompt),
    maxIterations: 100
);

How it works:

  • Start with variations of the initial prompt (population)
  • Evaluate how well each performs (fitness)
  • Select the best performers
  • Combine them (crossover) and introduce random changes (mutation)
  • Repeat for many generations

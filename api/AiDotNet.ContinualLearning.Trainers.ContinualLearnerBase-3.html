Remarks

For Algorithm Implementers: To create a new continual learning algorithm:

  1. Extend this base class
  2. Implement the abstract TrainOnTask(IDataset<T, TInput, TOutput>, IDataset<T, TInput, TOutput>?, int) method with your algorithm-specific training logic
  3. Pass an IContinualLearningStrategy<T, TInput, TOutput> that handles algorithm-specific regularization
  4. All shared functionality (evaluation, memory management, events) is handled automatically

Common Patterns:

  • Regularization-based (EWC, SI): Override TrainOnTask to add regularization loss
  • Replay-based (GEM, A-GEM): Use MemoryBuffer and override gradient computation
  • Distillation-based (LwF): Use the strategy's teacher model for knowledge transfer

Reference: De Lange et al. "A Continual Learning Survey: Defying Forgetting" (2021)

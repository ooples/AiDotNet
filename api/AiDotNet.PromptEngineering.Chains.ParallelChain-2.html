Remarks

A parallel chain runs multiple independent operations simultaneously on the same input, then combines their results using a reducer function.

For Beginners: Runs multiple operations at the same time for speed.

Example:

var chain = new ParallelChain<string, AnalysisReport>("ProductAnalysis", "Analyzes product from multiple angles");

// Add parallel branches
chain.AddBranch("sentiment", text => AnalyzeSentiment(text));
chain.AddBranch("keywords", text => ExtractKeywords(text));
chain.AddBranch("entities", text => ExtractEntities(text));

// Set how to combine results
chain.SetReducer((results) => new AnalysisReport
{
    Sentiment = (string)results["sentiment"],
    Keywords = (List<string>)results["keywords"],
    Entities = (List<string>)results["entities"]
});

// Run chain - all branches execute simultaneously
var result = chain.Run("Great product! Love the fast delivery from Amazon.");

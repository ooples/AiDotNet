Remarks

Backend Selection Order:

  1. CUDA (NVIDIA GPUs with NVRTC)
  2. OpenCL (AMD, Intel, NVIDIA via universal driver)
  3. HIP (AMD GPUs with ROCm SDK)

Use GetAvailableBackends() to check which GPU backends are available, and GetDiagnosticReport() for comprehensive troubleshooting information.

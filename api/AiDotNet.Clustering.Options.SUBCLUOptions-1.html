Remarks

SUBCLU (SUBspace CLUstering) is a density-connected subspace clustering algorithm that uses DBSCAN as its base clustering method and the monotonicity of density-connected clusters to efficiently prune the subspace search.

For Beginners: SUBCLU finds clusters hiding in feature subsets: - Uses DBSCAN's concept of density-connected clusters - Exploits the fact that if there's no cluster in 2D, there can't be one in 3D+ - This pruning makes it much faster than testing all possible subspaces

Great for datasets with many features where clusters only exist in some dimensions.

Properties

Epsilon

Gets or sets the epsilon radius for DBSCAN density computation.

public double Epsilon { get; set; }

Property Value

double

The neighborhood radius. Default is 0.5.

Remarks

For Beginners: This defines what "close" means: - Smaller epsilon: Only very nearby points are neighbors (tighter clusters) - Larger epsilon: Points farther apart are still neighbors (looser clusters)

MinPoints

Gets or sets the minimum points required for a dense region.

public int MinPoints { get; set; }

Property Value

int

Minimum neighborhood size. Default is 5.

Remarks

For Beginners: How many neighbors make a "core" point: - Higher MinPoints: Needs more points to form clusters (less noise sensitive) - Lower MinPoints: Easier to form clusters (more noise sensitive)

Rule of thumb: Set to dimensionality + 1 or higher.

Remarks

For Beginners: CoreSet is a diversity-based strategy that aims to select samples that best represent the entire dataset. It's like choosing a small set of "core" points that cover the data space well.

How CoreSet Works:

  1. Extract feature representations of all samples
  2. Start with labeled samples as initial "covered" points
  3. Iteratively select unlabeled points that maximize minimum distance to covered set
  4. This ensures selected points spread across the feature space

Distance Metrics:

  • Euclidean: Standard L2 distance in feature space
  • Cosine: Angle-based distance (1 - cosine similarity)
  • Manhattan: L1 distance for sparse features

When to Use:

  • Want diversity rather than just uncertainty
  • Data has meaningful geometric structure
  • Concerned about sample redundancy

Reference: Sener and Savarese "Active Learning for Convolutional Neural Networks: A Core-Set Approach" (ICLR 2018)

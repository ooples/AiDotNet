Defines the type of node in an execution graph.

public enum ExecutionNodeType

Fields

Allocate = 0

Buffer allocation node. Allocates GPU memory for a tensor.

Attention = 12

Attention operation node. Multi-head attention, flash attention, etc.

Barrier = 5

Synchronization barrier node. Forces all prior operations to complete before continuing.

Convolution = 13

Convolution operation node. 2D/3D convolutions.

Copy = 9

Buffer copy node. Copies data between GPU buffers.

Deallocate = 8

Buffer deallocation node. Releases GPU memory.

FusedKernel = 4

Fused kernel execution node. Runs a pre-fused sequence of kernels (e.g., GEMM+Bias+ReLU).

Kernel = 3

Single kernel execution node. Runs one GPU kernel.

Normalization = 11

Normalization operation node. Batch norm, layer norm, etc.

RecordEvent = 6

Event record node. Records an event that can be waited on by other streams.

Reduction = 10

Reduction operation node. Performs reduction operations (sum, max, etc.).

TransferD2H = 2

Device-to-host transfer node. Downloads data from GPU to CPU.

TransferH2D = 1

Host-to-device transfer node. Uploads data from CPU to GPU.

WaitEvent = 7

Event wait node. Waits for an event recorded by another stream.

Defines the interface for mapping features from a source domain to a target domain.

public interface IFeatureMapper<T>

Type Parameters

T

The numeric type used for calculations (e.g., double, float).

Remarks

For Beginners: A feature mapper is like a translator between two different languages. When you have data from one domain (source) and want to use it in another domain (target), the feature mapper transforms the data so it makes sense in the new context.

For example, if you trained a model on images (which might have thousands of features) and want to use that knowledge for text (which has different features), a feature mapper helps bridge that gap by finding a common representation.

Returns

T

A value between 0 and 1, where higher values indicate better mapping quality.

Remarks

For Beginners: This tells you how confident the mapper is in its translations. A high score (close to 1) means the mapper thinks it can translate well between the domains. A low score (close to 0) means the domains might be too different to map effectively.

Remarks

Bisecting K-Means is a divisive hierarchical clustering algorithm that starts with all points in one cluster and recursively bisects clusters until the desired number of clusters is reached.

For Beginners: Bisecting K-Means works by: 1. Starting with all data in one big cluster 2. Splitting the largest cluster into two using K-Means 3. Repeating step 2 until you have k clusters

Advantages over regular K-Means:

  • Often produces better clusters (more balanced)
  • Less sensitive to initialization
  • Builds a cluster hierarchy as a side effect

Gets or sets the number of bisection trials at each split.

public int NumBisectionTrials { get; set; }

Property Value

int

Number of times to try bisecting a cluster, keeping the best result. Higher values give better quality but are slower. Default is 5.

Remarks

For Beginners: When splitting a cluster, we try multiple times and keep the best split. This helps avoid bad splits caused by random initialization.

Interface for oracles (labeling providers) in active learning.

public interface IOracle<TInput, TOutput>

Type Parameters

TInput

The input data type.

TOutput

The output (label) data type.

Remarks

For Beginners: An oracle is the source of labels for unlabeled data. In real applications, this is typically a human expert. In experiments, it can be a simulator using ground-truth labels.

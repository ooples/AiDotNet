Remarks

This base class provides default implementations for batch distance computations that can be overridden by derived classes for optimized implementations.

For Beginners: This is the foundation for all distance metrics. It provides common code so each specific distance metric only needs to define how to compute the distance between two points.

The computed distance. Always non-negative for valid metrics.

Remarks

For Beginners: This calculates how far apart two points are.

For example, with Euclidean distance:

  • Points [0, 0] and [3, 4] have distance 5 (Pythagorean theorem)
  • Points [1, 1] and [1, 1] have distance 0 (same point)

Different metrics will give different values for the same pair of points.

ComputePairwise(Matrix<T>)

Computes the full pairwise distance matrix between all rows.

public virtual Matrix<T> ComputePairwise(Matrix<T> data)

Parameters

data Matrix<T>

A matrix where each row is a data point.

Returns

Matrix<T>

A symmetric matrix where element [i, j] is the distance between row i and row j. The diagonal is always zero.

Remarks

This operation has O(n^2) complexity where n is the number of rows. For large datasets, consider using spatial indexing structures like KD-Trees.

For Beginners: This creates a table showing the distance between every pair of points. It's like a mileage chart showing distances between cities.

The result is symmetric (distance A to B equals distance B to A) and the diagonal is all zeros (distance from a point to itself is zero).

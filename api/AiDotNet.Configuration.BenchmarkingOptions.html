Remarks

This options class is designed for use with PredictionModelBuilder.ConfigureBenchmarking(...) and PredictionModelResult.EvaluateBenchmarksAsync(...). Users specify which benchmark suites to run, and AiDotNet orchestrates the execution behind the scenes.

For Beginners: Benchmarking is how you measure model quality using standardized tests. You choose a suite (like GSM8K for math reasoning), and AiDotNet runs it and returns a report.

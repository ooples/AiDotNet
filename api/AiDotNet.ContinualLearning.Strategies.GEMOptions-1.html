For Beginners: These options control how GEM prevents forgetting by constraining gradients to not harm previous task performance.

double?

Remarks

Positive values allow small constraint violations for faster convergence.

Negative values require strict improvement on all tasks.

Reference: Lopez-Paz et al. used 0.5 in their experiments.

UseAGEM

Whether to use Averaged GEM (A-GEM) instead of full GEM. Default: false.

public bool? UseAGEM { get; set; }

Property Value

bool?

Remarks

A-GEM: Projects gradient onto the average reference gradient from a random sample of memory, instead of checking all task constraints individually.

Advantage: O(1) complexity vs O(t) for t tasks.

Disadvantage: Weaker guarantees but much faster.

Reference: Chaudhry et al. "Efficient Lifelong Learning with A-GEM" (2019)

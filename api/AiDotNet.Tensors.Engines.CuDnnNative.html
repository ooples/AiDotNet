Remarks

Performance Targets:

  • Convolution: ~15,000 GFLOPS (cuDNN optimized algorithms)
  • BatchNorm: Near memory-bandwidth limited
  • Pooling: Near memory-bandwidth limited
  • Attention: Tensor core accelerated (Volta+)

cuDNN Frontend Reference: https://github.com/NVIDIA/cudnn-frontend

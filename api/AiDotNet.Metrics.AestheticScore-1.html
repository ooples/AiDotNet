Remarks

Aesthetic Score uses CLIP embeddings trained on aesthetic preference data to predict how visually appealing an image is. This is commonly used in image generation to filter or rank outputs by aesthetic quality.

Typical aesthetic scores (1-10 scale): - >7: High aesthetic quality - 5-7: Average aesthetic quality - <5: Low aesthetic quality

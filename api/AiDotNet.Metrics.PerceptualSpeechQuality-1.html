Remarks

This is a simplified approximation of the ITU-T P.862 PESQ algorithm. For production use, consider using an official PESQ implementation.

PESQ scores range from -0.5 to 4.5, where higher values indicate better quality. - >4.0: Excellent quality - 3.5-4.0: Good quality - 3.0-3.5: Fair quality - <3.0: Poor quality

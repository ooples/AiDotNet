Namespace AiDotNet.Normalizers

Classes

BinningNormalizer<T, TInput, TOutput>

Represents a normalizer that uses data binning to transform values into discrete ranges.

DecimalNormalizer<T, TInput, TOutput>

Normalizes the data by dividing each value by the smallest multiple of 10 that is greater than the largest value.

GlobalContrastNormalizer<T, TInput, TOutput>

Normalizes data by adjusting its contrast globally based on the mean and standard deviation.

LogMeanVarianceNormalizer<T, TInput, TOutput>

Normalizes data by taking the logarithm and then applying mean-variance normalization.

LogNormalizer<T, TInput, TOutput>

Normalizes the data by taking the natural log of each value.

LpNormNormalizer<T, TInput, TOutput>

Normalizes vectors using the Lp-norm, dividing each element by the vector's p-norm.

MaxAbsScaler<T, TInput, TOutput>

Scales features to the range [-1, 1] by dividing by the maximum absolute value.

MeanVarianceNormalizer<T, TInput, TOutput>

Normalizes data by standardizing it to have zero mean and unit variance.

MinMaxNormalizer<T, TInput, TOutput>

Normalizes the data by

  1. Subtracting the minimum value from each value
  2. Dividing each value from step #1 by the absolute difference between the maximum and minimum values
NoNormalizer<T, TInput, TOutput>

A normalizer that does not modify the data but maintains the same interface as other normalizers.

NormalizerBase<T, TInput, TOutput>

Base class for normalizers that provides common functionality and implements the INormalizer interface.

QuantileTransformer<T, TInput, TOutput>

Transforms features to follow a uniform or normal distribution using quantile information.

RobustScalingNormalizer<T, TInput, TOutput>

Normalizes data using robust scaling based on median and interquartile range (IQR).

ZScoreNormalizer<T, TInput, TOutput>

Normalizes the data by subtracting the mean from each value and dividing by the standard deviation.

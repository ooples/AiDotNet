Remarks

This optimizer runs multiple optimization strategies and combines their results, using voting, averaging, or selection to determine the best prompt.

For Beginners: Like getting multiple opinions and combining them.

Example:

var optimizer = new EnsembleOptimizer<double>(
    new DiscreteSearchOptimizer<double>(),
    new BeamSearchOptimizer<double>(),
    new GeneticOptimizer<double>()
);

var optimized = optimizer.Optimize(
    initialPrompt: "Classify sentiment:",
    evaluationFunction: prompt => EvaluateAccuracy(prompt),
    maxIterations: 100  // Divided among strategies
);

How it works:

  • Run multiple optimizers independently
  • Collect their best results
  • Pick the overall best (or combine them)

Benefits:

  • More robust than single strategy
  • Different strategies find different optima
  • Better coverage of solution space

Namespace
AiDotNet.Tensors.Engines

Extension interface for sparse tensor operations.

public interface ISparseEngine

Remarks

ISparseEngine provides operations for sparse tensors that are too memory-intensive to store as dense arrays. Common in graph neural networks, recommendation systems, and NLP (bag-of-words) applications.

For Beginners: Sparse tensors store only non-zero values.

Example: A 1000x1000 matrix with only 100 non-zero values:

  • Dense storage: 1,000,000 values (8MB for doubles)
  • Sparse storage: ~300 values (indices + values = ~2.4KB)

SpMM = Sparse Matrix × Dense Matrix SpMV = Sparse Matrix × Dense Vector

Parameters

The sparse matrix A in CSR format.

dense Vector<T>

The dense vector x.

Returns

Vector<T>

The result vector y.

Type Parameters

T

The numeric type.

Remarks

For Beginners: This multiplies a sparse matrix by a dense vector. Only non-zero elements of the sparse matrix contribute to the computation, making this much faster than dense matrix-vector multiplication for sparse matrices.

Time complexity: O(nnz) where nnz is the number of non-zeros in the sparse matrix.

Exceptions

ArgumentException

Thrown when dimensions are incompatible.

Remarks

Video enhancers take degraded or low-quality video and produce improved versions. Common enhancement tasks include super-resolution (upscaling), denoising, stabilization, and frame interpolation.

For Beginners: Video enhancement is like photo editing, but for videos. These models can: - Make blurry videos sharper (super-resolution) - Remove grain/noise from old or low-light videos - Smooth out shaky camera footage (stabilization) - Make choppy videos smoother (frame interpolation)

The enhanced video has the same content but looks much better!

Example:

var enhancer = new VideoSuperResolution<double>(new VideoEnhancementOptions<double>
{
    ScaleFactor = 4  // 4x upscaling
});

// Upscale a 480p video to 1920p (4x larger)
var hdVideo = enhancer.Enhance(lowResVideo);

ScaleFactor

Gets the scale factor for spatial enhancement (e.g., 2x, 4x upscaling).

int ScaleFactor { get; }

Property Value

int

Remarks

A value of 1 means no spatial scaling (same resolution output). A value of 4 means the output is 4x larger in both width and height.

For Beginners: This tells you how much bigger the output video will be. ScaleFactor = 2 means a 720x480 video becomes 1440x960. ScaleFactor = 4 means a 720x480 video becomes 2880x1920.

SupportsRealTime

Gets whether this enhancer can process video in real-time.

bool SupportsRealTime { get; }

Property Value

bool

Remarks

Real-time means the model can process frames fast enough to keep up with live video playback (typically 24-30 frames per second).

For Beginners: If this is true, you can use the enhancer for live video. If false, it's better suited for batch processing of recorded videos.

TemporalScaleFactor

Gets the temporal scale factor for frame rate enhancement.

int TemporalScaleFactor { get; }

Property Value

int

Remarks

A value of 1 means no frame rate change. A value of 2 means the output has 2x more frames (e.g., 30fps becomes 60fps).

For Beginners: This tells you how many extra frames will be added. TemporalScaleFactor = 2 doubles the frame rate for smoother playback.

For Beginners: This is the main method to improve your video. Just pass in your video and get back an enhanced version!

Remarks

CpuSparseEngine provides efficient sparse matrix operations using standard algorithms optimized for CPU execution. All operations work with the SparseTensor type which supports COO, CSR, and CSC storage formats.

For Beginners: This class does the actual work for sparse operations on the CPU. It's used when you don't have a GPU or when working with custom numeric types.

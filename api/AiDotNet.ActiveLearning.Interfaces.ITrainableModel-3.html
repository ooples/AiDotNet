Interface for models that support training on datasets.

public interface ITrainableModel<T, TInput, TOutput>

Type Parameters

T

The numeric type used for calculations.

TInput

The input data type.

TOutput

The output data type.

Remarks

For Beginners: A trainable model is one that can learn from data. The training process adjusts the model's internal parameters to improve its predictions based on the provided examples.

Common Uses:

  • Active learning iteratively trains models on newly labeled samples
  • Query by Committee trains multiple models on bootstrap samples
  • Continual learning trains models on sequential tasks

Reset()

void Reset()

Remarks

Clears all learned parameters, effectively creating a fresh model. Useful when retraining from scratch or for cross-validation.

Parameters

dataset IDataset<T, TInput, TOutput>

The dataset containing input-output pairs for training.

Remarks

For Beginners: This method updates the model's internal parameters by learning patterns from the training data. After training, the model should produce better predictions for similar inputs.

Train(IDataset<T, TInput, TOutput>, int)

Trains the model for a specified number of epochs.

void Train(IDataset<T, TInput, TOutput> dataset, int epochs)

Parameters

dataset IDataset<T, TInput, TOutput>

The dataset containing input-output pairs for training.

epochs int

The number of training epochs (passes through the data).

Remarks

For Beginners: An epoch is one complete pass through all training data. More epochs generally lead to better learning, but too many can cause overfitting (memorizing the training data rather than learning general patterns).

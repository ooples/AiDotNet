Remarks

KMeans is a centroid-based clustering algorithm that partitions n observations into k clusters where each observation belongs to the cluster with the nearest centroid.

For Beginners: KMeans works by: 1. Choosing k initial center points 2. Assigning each data point to its nearest center 3. Recalculating centers as the mean of assigned points 4. Repeating steps 2-3 until centers stabilize

The key settings are:

  • Number of clusters (k): How many groups to find
  • Initialization method: How to choose starting centers
  • Number of runs: How many times to try different starting points

InitialCenters

Gets or sets custom initial cluster centers.

public Matrix<T>? InitialCenters { get; set; }

Property Value

Matrix<T>

Matrix where each row is a cluster center, or null to use InitMethod.

Remarks

When set, this overrides the InitMethod setting. The matrix must have NumClusters rows and the same number of columns as the input data.

NumClusters

Gets or sets the number of clusters to find.

public int NumClusters { get; set; }

Property Value

int

The number of clusters. Must be positive.

Remarks

For Beginners: This is the "k" in KMeans - how many groups you want the algorithm to find. If you're not sure, try different values and compare results using the elbow method or silhouette score.

PrecomputeDistances

Gets or sets whether to precompute distances.

public PrecomputeOption PrecomputeDistances { get; set; }

Property Value

PrecomputeOption

When true, computes full distance matrix (faster but uses O(n*k) memory). When false, computes distances on-the-fly (slower but memory efficient). Default is Auto.

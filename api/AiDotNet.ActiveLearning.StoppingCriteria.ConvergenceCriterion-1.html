Remarks

For Beginners: This criterion analyzes the learning curve (how performance changes with more data) and stops when adding more data shows diminishing returns.

How It Works:

  1. Fits a power law or logarithmic curve to performance history
  2. Extrapolates expected improvement from more samples
  3. Stops when expected improvement is below threshold

Key Insight: Learning curves often follow a power law: performance = a - b*n^(-c) where n is the number of samples. As n grows, improvement decreases.

When to Use:

  • Want to optimize cost/benefit of labeling
  • Have enough history to estimate the curve
  • Model performance is expected to follow standard learning curves

Reference: Figueroa et al. "Predicting sample size required for classification performance" (BMC 2012)

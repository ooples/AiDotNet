Interface for models that can extract feature representations from inputs.

public interface IFeatureExtractor<T, TInput>

Type Parameters

T

The numeric type used for calculations.

TInput

The input data type.

Remarks

For Beginners: Feature extraction converts raw input data into a numerical representation (feature vector) that captures the important characteristics of the input. This is useful for tasks like clustering, similarity comparisons, and active learning strategies that need to measure diversity.

Common Uses:

  • BADGE active learning strategy uses gradient embeddings as features
  • CoreSet strategy uses feature distances for diversity sampling
  • Similarity-based sample selection

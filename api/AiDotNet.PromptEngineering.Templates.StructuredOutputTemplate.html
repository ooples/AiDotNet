Remarks

This template helps ensure consistent, parseable output by providing format specifications, schemas, and examples of the expected structure.

For Beginners: Gets AI to output data in a specific format.

Example:

var template = new StructuredOutputTemplate(
    OutputFormat.Json,
    schema: @"{
        ""name"": ""string"",
        ""age"": ""number"",
        ""email"": ""string""
    }"
);

var prompt = template.WithTask("Extract user info from: John Doe, 30, john@email.com")
    .Format(new Dictionary<string, string>());

Supported formats:

  • JSON for APIs and data processing
  • XML for configuration and interchange
  • CSV for tabular data
  • Markdown for documentation
  • Custom formats with user-defined schemas

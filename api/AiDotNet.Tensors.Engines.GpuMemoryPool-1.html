Remarks

GpuMemoryPool reduces GPU memory allocation overhead by reusing buffers across operations. Buffers are organized into size buckets for efficient reuse.

Phase B: US-GPU-002 - Memory Buffer Pooling

Benefits:

  • 5-10x reduction in allocation overhead
  • Prevents memory fragmentation
  • Thread-safe for concurrent operations
  • Automatic buffer growth when pool exhausted

Size buckets: 1K, 10K, 100K, 1M, 10M elements

Remarks

If a buffer is available in the pool, it is reused. Otherwise, a new buffer is allocated. The returned buffer may be larger than requested to fit the bucket size.

IMPORTANT: You must call Return(MemoryBuffer1D<T, Dense>) when done with the buffer to return it to the pool.

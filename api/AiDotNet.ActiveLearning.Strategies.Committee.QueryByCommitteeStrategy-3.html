Remarks

For Beginners: QBC uses a "committee" of diverse models to identify samples where the models disagree most. High disagreement indicates uncertainty about the true label.

How QBC Works:

  1. Train multiple models (committee) on the same data
  2. For each unlabeled sample, collect predictions from all committee members
  3. Measure disagreement using vote entropy or KL divergence
  4. Select samples with highest disagreement for labeling

Disagreement Measures:

  • Vote Entropy: Entropy of the vote distribution
  • KL Divergence: Average divergence from consensus
  • Soft Vote Variance: Variance of probability predictions

When to Use:

  • Can train multiple diverse models
  • Want to capture model uncertainty through ensemble disagreement
  • Classification problems with discrete predictions

Reference: Seung et al. "Query by Committee" (1992)

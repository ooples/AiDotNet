Remarks

These extensions enable seamless integration between trained models and the serving infrastructure. Use these methods to convert a PredictionModelResult into a format suitable for the REST API.

For Beginners: After training a model, you need to convert it to a serving format.

Example:

// Train a model using PredictionModelBuilder
var trainedModel = builder.Build(trainingData, validationData);

// Convert to servable model for the REST API
var servableModel = trainedModel.ToServableModel("my-model", 10, 1);

// Register with the model repository
modelRepository.LoadModel("my-model", servableModel);

Remarks

For Beginners: BADGE (Batch Active learning by Diverse Gradient Embeddings) is a state-of-the-art method that combines uncertainty and diversity. It uses gradient embeddings - vectors derived from the model's gradients - to represent samples.

How It Works:

  1. Compute gradient embeddings for all unlabeled samples
  2. Use k-means++ initialization to select diverse embeddings
  3. The gradient magnitude captures uncertainty
  4. k-means++ ensures diversity in gradient space

Why Gradient Embeddings?

  • Gradients are large for uncertain predictions (model wants to change)
  • Similar gradients indicate redundant information
  • Combines uncertainty and diversity in a principled way

Reference: Ash et al. "Deep Batch Active Learning by Diverse, Uncertain Gradient Lower Bounds" (ICLR 2020)

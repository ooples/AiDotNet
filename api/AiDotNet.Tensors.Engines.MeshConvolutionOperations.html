Remarks

This class implements vectorized mesh convolution algorithms including: - Spiral convolution for irregular mesh connectivity - Diffusion convolution using heat equation on surfaces - Mesh Laplacian computation for geometry-aware processing

All operations are optimized using SIMD vectorization via System.Numerics.Vector where applicable, and parallel processing for batch operations.

Remarks

Implements diffusion convolution using Taylor series approximation of exp(-t*L): exp(-t*L) ≈ I - t*L + (t²/2)*L² - (t³/6)*L³ + ...

Using 4 terms provides good accuracy for reasonable diffusion times.

T

Remarks

The algorithm: 1. For each vertex, gather features from neighbors in spiral order 2. Concatenate gathered features into a single vector 3. Apply linear transformation (matrix multiply with weights) 4. Add bias

Uses SIMD vectorization for the matrix multiplication and parallel processing across vertices for optimal performance.

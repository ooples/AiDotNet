HipBackend

HIP backend for AMD GPUs with real MFMA (Matrix Core) acceleration. Automatically selects the optimal kernel based on GPU architecture:

  • CDNA (MI100/200/300): Full MFMA with wave64
  • RDNA3 (RX 7000): WMMA with wave32
  • RDNA2 (RX 6000): Optimized scalar with wave32

Remarks

WER measures the accuracy of automatic speech recognition (ASR) systems by computing the edit distance between the predicted transcription and the reference transcription.

Formula: WER = (Substitutions + Insertions + Deletions) / Number of words in reference

Typical WER values: - <5%: Excellent (human-level performance) - 5-10%: Very good - 10-20%: Good - 20-30%: Acceptable - >30%: Poor

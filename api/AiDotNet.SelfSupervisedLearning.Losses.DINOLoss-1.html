Remarks

For Beginners: DINO loss is a cross-entropy loss between student and teacher outputs, where the teacher is an EMA of the student. It uses centering and sharpening to prevent collapse.

Key components:

  • Sharpening: Lower temperature for teacher outputs (default: 0.04)
  • Student temperature: Higher temperature for student (default: 0.1)
  • Centering: Subtract running mean from teacher outputs to prevent collapse

Loss formula:

L = -Σ_crops Σ_k P_t(k) * log(P_s(k))
where P_t = softmax((z_t - c) / τ_t) and P_s = softmax(z_s / τ_s)

Reference: Caron et al., "Emerging Properties in Self-Supervised Vision Transformers" (ICCV 2021)

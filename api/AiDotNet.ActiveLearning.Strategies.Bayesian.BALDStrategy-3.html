Remarks

For Beginners: BALD uses Bayesian principles to select samples that would provide the most information about the model's parameters. It measures the mutual information between predictions and model parameters.

How BALD Works:

  1. Use Monte Carlo Dropout to sample from the posterior
  2. Compute predictive entropy H[y|x,D]
  3. Compute expected posterior entropy E[H[y|x,w]]
  4. Select samples with highest mutual information: I[y,w|x,D] = H[y|x,D] - E[H[y|x,w]]

When to Use:

  • Model supports dropout or other uncertainty estimation
  • Want to distinguish between model uncertainty and data noise
  • Deep learning models with dropout layers

Reference: Gal et al. "Deep Bayesian Active Learning with Image Data" (ICML 2017)

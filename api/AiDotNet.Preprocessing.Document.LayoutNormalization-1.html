Remarks

LayoutNormalization provides utilities for normalizing document layouts to standard sizes and aspect ratios for consistent model input.

For Beginners: Different documents have different sizes and proportions. This tool standardizes them for AI models:

  • Resize to target dimensions
  • Preserve aspect ratio options
  • Handle padding and cropping
  • Normalize orientation

Key features:

  • Multiple normalization strategies
  • Aspect ratio preservation
  • Smart padding and cropping
  • Batch processing support

Example usage:

var normalizer = new LayoutNormalization<float>();
var normalized = normalizer.Process(document, targetWidth: 224, targetHeight: 224);

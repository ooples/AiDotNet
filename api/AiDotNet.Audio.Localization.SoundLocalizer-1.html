Remarks

Estimates the direction of arrival (DOA) of sound sources using multiple microphones. Supports various algorithms: GCC-PHAT, MUSIC, SRP-PHAT.

For Beginners: Sound localization finds where sounds come from:

  • Uses 2+ microphones to detect time differences in sound arrival
  • Calculates direction (azimuth/elevation) or 3D position of sound sources
  • Works like how human ears localize sounds

Usage:

// Define microphone array geometry (2 mics, 10cm apart)
var micPositions = new double[,] { { -0.05, 0, 0 }, { 0.05, 0, 0 } };
var localizer = new SoundLocalizer<float>(micPositions);

// Audio from each microphone
var audioChannels = new[] { LoadAudio("mic1.wav"), LoadAudio("mic2.wav") };
var result = localizer.Localize(audioChannels);

Console.WriteLine($"Sound from {result.AzimuthDegrees:F1} degrees");

Remarks

For Beginners: Sometimes you want to stop based on multiple conditions. For example, stop if you run out of budget OR if performance plateaus. This composite criterion lets you combine multiple criteria with AND/OR logic.

Combination Modes:

  • Any (OR): Stop if ANY criterion says to stop
  • All (AND): Stop only if ALL criteria say to stop

Common Combinations:

  • Budget OR Plateau: Stop when budget exhausted or performance plateaus
  • Budget AND Convergence: Stop only after both conditions met
  • Time OR (Budget AND Plateau): Complex nested logic

Example Usage:

var composite = new CompositeCriterion<double>(CombinationMode.Any);
composite.AddCriterion(new BudgetExhaustedCriterion<double>(100));
composite.AddCriterion(new PerformancePlateauCriterion<double>());
composite.AddCriterion(new TimeBudgetCriterion<double>(TimeSpan.FromHours(2)));

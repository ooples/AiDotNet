Namespace
AiDotNet.Tensors.Interfaces

Defines vectorized (array/span-based) operations for numeric types that can be SIMD-optimized.

public interface IVectorizedOperations<T>

Type Parameters

T

The numeric data type for the operations.

Remarks

This interface provides batch operations on arrays or spans of numeric values. Implementations can use hardware acceleration (SIMD via TensorPrimitives) when available, or fall back to sequential loops for unsupported types.

For Beginners: While INumericOperations<T> handles single-value operations (like adding two numbers), this interface handles operations on entire arrays at once.

Modern CPUs can perform the same operation on multiple values simultaneously using SIMD (Single Instruction Multiple Data). For example, adding two arrays of 8 floats might complete in a single CPU instruction instead of 8 separate additions.

Performance Characteristics (with AVX2): - Element-wise operations (Add, Multiply): 5-10x speedup - Reductions (Sum, Max, Min): 8-12x speedup - Transcendentals (Exp, Log, Tanh): 3-6x speedup - Dot product: 10-15x speedup on large vectors

Remarks

Key Features:

  • Works on ALL .NET versions (4.6.2, 4.7.1, net8.0, etc.)
  • No managed GPU runtime dependency - pure P/Invoke
  • Double-buffered GEMM for compute/memory overlap
  • Fused operations (GEMM+Bias+Activation)
  • Bank-conflict-free shared memory

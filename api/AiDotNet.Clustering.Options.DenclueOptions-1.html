Remarks

DENCLUE (DENsity-based CLUstEring) uses kernel density estimation to find clusters. Points are attracted to density maxima using gradient ascent, and clusters are formed by grouping points that converge to the same attractor.

For Beginners: DENCLUE sees data as a "landscape" where dense areas form "mountains" (attractors).

Imagine pouring water on a terrain:

  • Water flows downhill to valleys
  • In DENCLUE, points "flow uphill" to peaks
  • Points reaching the same peak form a cluster

Key parameters:

  • Bandwidth (h): Controls how wide the "mountains" are
    • Small h: Many narrow peaks (more clusters)
    • Large h: Few wide peaks (fewer clusters)
  • MinDensity: Minimum density for a peak to be a cluster center

AttractorMergeThreshold

public double AttractorMergeThreshold { get; set; }

Property Value

double

Attractor merge threshold. Default is 0.1.

Remarks

Attractors closer than this threshold are merged into a single cluster. This helps avoid creating many tiny clusters from nearly identical attractors.

Bandwidth

Gets or sets the bandwidth parameter for the Gaussian kernel.

public double Bandwidth { get; set; }

Property Value

double

Bandwidth (h). Default is 1.0.

Remarks

Controls the width of the Gaussian kernel used for density estimation. Smaller values make the density function more peaked; larger values smooth it out.

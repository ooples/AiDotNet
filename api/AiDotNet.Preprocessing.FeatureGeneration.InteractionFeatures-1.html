Remarks

InteractionFeatures creates new features by multiplying pairs of existing features. Unlike PolynomialFeatures with degree=2, this only produces interaction terms, not squared terms.

For features [a, b, c], this produces: [ab, ac, bc]

For Beginners: Interaction features capture combined effects: - If both "age" and "income" matter together (not just separately) - Creating age Ã— income might help the model - This is simpler than full polynomial features (no squared terms)

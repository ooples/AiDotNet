Remarks

Speculative decoding accelerates text generation by using a smaller draft model to generate candidate tokens that are then verified by the target model.

For Beginners: Think of speculative decoding like having a fast assistant who suggests multiple words at once, which you then verify. Instead of generating one token at a time (slow), we generate several candidates quickly and verify them in parallel (fast).

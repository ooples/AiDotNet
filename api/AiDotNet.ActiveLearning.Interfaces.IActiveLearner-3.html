Remarks

For Beginners: An active learner is a machine learning system that actively selects which data points should be labeled. Instead of labeling all data, it strategically chooses the most informative samples to learn from.

Active Learning Workflow:

  1. Start with a small labeled dataset and large unlabeled pool
  2. Train the model on labeled data
  3. Use a query strategy to select informative unlabeled samples
  4. Get labels for selected samples (from oracle/human)
  5. Add newly labeled samples to training set
  6. Repeat until stopping criterion is met

When to Use Active Learning:

  • Labeling is expensive (requires human experts)
  • Large amounts of unlabeled data available
  • Model can benefit from strategic sample selection

Remarks

For Beginners: Active learning is a machine learning paradigm where the algorithm actively selects which data points should be labeled by an oracle (human expert). This is particularly useful when labeling data is expensive or time-consuming.

How Active Learning Works:

  1. Start with a small labeled dataset and a large unlabeled pool
  2. Train a model on the labeled data
  3. Use a query strategy to select the most informative unlabeled samples
  4. Request labels for these samples from an oracle
  5. Add newly labeled samples to the training set and repeat

Key Concepts:

  • Query Strategy: The method used to select which samples to label next
  • Uncertainty Sampling: Select samples where the model is most uncertain
  • Query By Committee: Use an ensemble to find disagreement
  • Diversity Sampling: Select samples that are diverse in feature space

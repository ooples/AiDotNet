Remarks

This service runs during application startup and loads models specified in the ServingOptions.StartupModels configuration. Models are loaded as PredictionModelResult instances to maintain the facade pattern and include all configured optimizations.

For Beginners: This service automatically loads models when your server starts.

Configure startup models in appsettings.json:

{
  "ServingOptions": {
    "StartupModels": [
      { "Name": "my-model", "Path": "models/my-model.aidotnet", "NumericType": "double" }
    ]
  }
}

Benefits:

  • Models are ready immediately when the server starts
  • No cold start latency for first prediction
  • Validates models exist and load correctly at startup

Remarks

Backend Selection Strategy:

  • NVIDIA: CUDA (primary) → OpenCL (fallback)
  • AMD: HIP/ROCm (if available) → OpenCL (fallback)
  • Intel: OpenCL

The factory automatically detects the GPU vendor and selects the most performant backend. For NVIDIA GPUs, CUDA typically provides 10-20% better performance than OpenCL. For AMD GPUs, HIP/ROCm can provide better performance when available.

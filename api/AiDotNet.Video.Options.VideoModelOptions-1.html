Remarks

All options use nullable properties with industry-standard defaults applied internally. This allows zero-configuration usage while still enabling full customization.

For Beginners: You don't need to set any of these options to get started! The model will use sensible defaults for everything. Only change options if you know you need something different.

Example - Using defaults:

var model = new VideoSuperResolution<double>();  // All defaults

Example - Customizing some options:

var model = new VideoSuperResolution<double>(new VideoEnhancementOptions<double>
{
    ScaleFactor = 4,      // Custom: 4x upscaling
    LearningRate = 0.001  // Custom: faster learning
    // Everything else uses defaults
});

DropoutRate

Gets or sets the dropout rate for regularization.

public double? DropoutRate { get; set; }

Property Value

double?

Remarks

Default: 0.1 (10% dropout). Helps prevent overfitting. Set to 0 to disable.

For Beginners: Dropout randomly ignores some neurons during training, which helps the model generalize better to new data instead of memorizing training data.

HiddenDimension

public int? HiddenDimension { get; set; }

int?

Remarks

Default: 768 (industry standard for vision transformers). Larger values can learn more complex patterns but use more memory.

For Beginners: This controls how much "thinking capacity" the model has. The default works well for most cases. Only increase if you have lots of GPU memory and need to learn very complex patterns.

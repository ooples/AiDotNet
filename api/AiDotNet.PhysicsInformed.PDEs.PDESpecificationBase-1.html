Remarks

For Beginners: A Partial Differential Equation (PDE) describes how a quantity changes with respect to multiple variables (like space and time). This base class provides the foundation for implementing any PDE that can be solved using Physics-Informed Neural Networks.

All PDE implementations should inherit from this class rather than implementing IPDESpecification<T> directly. This ensures consistent behavior and provides access to common helper methods.

To create a new PDE, override the abstract members:

  • ComputeResidual(T[], T[], PDEDerivatives<T>) - Calculate how much the PDE is violated
  • InputDimension - Number of independent variables (space + time)
  • OutputDimension - Number of dependent variables (solution components)
  • Name - Human-readable name of the PDE

ComputeResidual(T[], T[], PDEDerivatives<T>)

Computes the PDE residual at the given point. The residual is how much the PDE equation is violated at that point. For a true solution, the residual should be zero everywhere.

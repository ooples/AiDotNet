Remarks

For Beginners: This is the most common dataset implementation. It stores all data in memory, making it fast to access but limited by available RAM. Suitable for most datasets that fit in memory.

Features:

  • Fast random access to any sample
  • Immutable operations return new dataset instances
  • Supports labeled and unlabeled data
  • Thread-safe read operations

Usage:

// Create a labeled dataset
var inputs = new[] { new double[] { 1, 2 }, new double[] { 3, 4 } };
var outputs = new[] { 0, 1 };
var dataset = new InMemoryDataset<double, double[], int>(inputs, outputs);

// Create an unlabeled dataset
var unlabeled = InMemoryDataset<double, double[], int>.CreateUnlabeled(inputs);

`Models & Types` · `AiDotNet.RetrievalAugmentedGeneration.DocumentStores`

ChromaDB-based document store designed for simplicity and developer experience.

## How It Works

ChromaDB is an open-source vector database that emphasizes ease of use while maintaining
high performance for similarity search operations.

## Methods

| Method | Summary |
|:-----|:--------|
| `Clear` | Removes all documents from the ChromaDB collection and recreates it. |
| `GetAllCore` | Core logic for retrieving all documents from the ChromaDB collection. |
| `RemoveCore(String)` | Core logic for removing a document from the ChromaDB collection. |

`Models & Types` · `AiDotNet.RetrievalAugmentedGeneration.DocumentStores`

Azure Cognitive Search-inspired document store with field-based indexing and search capabilities.
Provides in-memory simulation of Azure Search features including field-level search and faceted filtering.

## Methods

| Method | Summary |
|:-----|:--------|
| `Clear` | Removes all documents from the index and clears all inverted indices. |
| `GetAllCore` | Core logic for retrieving all documents in the index. |

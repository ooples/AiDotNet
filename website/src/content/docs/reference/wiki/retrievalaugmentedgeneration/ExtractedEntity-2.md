`Models & Types` · `AiDotNet.RetrievalAugmentedGeneration.NER`

Represents an entity extracted from text.

## Properties

| Property | Summary |
|:-----|:--------|
| `Confidence` | Confidence score for this entity (0.0 to 1.0). |
| `EndIndex` | Ending token index in the original text. |
| `StartIndex` | Starting token index in the original text. |
| `Text` | The entity text. |
| `Type` | The entity type (PERSON, ORGANIZATION, LOCATION, DATE). |

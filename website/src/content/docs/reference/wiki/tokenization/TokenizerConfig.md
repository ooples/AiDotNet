`Options & Configuration` · `AiDotNet.Tokenization.HuggingFace`

Configuration for HuggingFace tokenizers.

## Properties

| Property | Summary |
|:-----|:--------|
| `AdditionalSpecialTokens` | Gets or sets additional special tokens. |
| `DoLowerCase` | Gets or sets whether to lowercase input. |
| `MergesFile` | Gets or sets the merges file (for BPE). |
| `ModelMaxLength` | Gets or sets the model max length. |
| `ModelType` | Gets or sets the model type. |
| `TokenizerClass` | Gets or sets the tokenizer type. |
| `UnkToken` | Gets or sets special tokens. |
| `VocabFile` | Gets or sets the vocabulary file. |

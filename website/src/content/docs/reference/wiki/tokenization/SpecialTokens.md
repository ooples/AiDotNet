`Models & Types` · `AiDotNet.Tokenization.Models`

Represents special tokens used by tokenizers.

## Properties

| Property | Summary |
|:-----|:--------|
| `AdditionalSpecialTokens` | Gets or sets additional special tokens. |
| `BosToken` | Gets or sets the beginning of sequence token. |
| `ClsToken` | Gets or sets the classification token (start of sequence). |
| `EosToken` | Gets or sets the end of sequence token. |
| `MaskToken` | Gets or sets the mask token. |
| `PadToken` | Gets or sets the padding token. |
| `SepToken` | Gets or sets the separation token. |
| `UnkToken` | Gets or sets the unknown token. |

## Methods

| Method | Summary |
|:-----|:--------|
| `Bert` | Creates BERT-style special tokens. |
| `Default` | Creates default special tokens (BERT-style). |
| `GetAllSpecialTokens` | Gets all special tokens as a list. |
| `Gpt` | Creates GPT-style special tokens. |
| `T5` | Creates T5-style special tokens. |

`Helpers & Utilities` · `AiDotNet.Clustering.Hierarchical`

Represents a node in the bisection hierarchy tree.

## Properties

| Property | Summary |
|:-----|:--------|
| `ClusterId` | The cluster ID of this node. |
| `Inertia` | Inertia (sum of squared distances to center) of this cluster. |
| `ParentId` | The parent cluster ID (-1 for root). |
| `Size` | Number of points in this cluster. |

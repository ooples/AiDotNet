`Models & Types` · `AiDotNet.Clustering.Subspace`

Represents a cluster discovered in a subspace.

## Properties

| Property | Summary |
|:-----|:--------|
| `ClusterId` | Unique identifier for this cluster. |
| `Dimensionality` | Dimensionality of the subspace. |
| `Dimensions` | The dimensions (feature indices) that define this subspace. |
| `NumUnits` | The number of dense units in this cluster. |
| `Points` | The indices of points belonging to this cluster. |
| `Size` | Number of points in this cluster. |

`Options & Configuration` · `AiDotNet.Clustering.AutoML`

Configuration options for ClusteringAutoML.

## Properties

| Property | Summary |
|:-----|:--------|
| `FuzzinessRange` | Range of fuzziness values for Fuzzy C-Means. |
| `KRange` | Range of K values to try for K-based algorithms. |
| `MaxIterationsPerTrial` | Maximum iterations per trial. |
| `MinPtsRange` | Range of MinPoints values for DBSCAN. |
| `RandomSeed` | Random seed for reproducibility. |
| `TryAutoK` | Whether to try Auto-K algorithms (X-Means, G-Means). |
| `TryDBSCAN` | Whether to try DBSCAN algorithm. |
| `TryFuzzyCMeans` | Whether to try Fuzzy C-Means algorithm. |
| `TryGaussianMixture` | Whether to try Gaussian Mixture algorithm. |
| `TryKMeans` | Whether to try KMeans algorithm. |

`Models & Types` · `AiDotNet.Statistics`

Represents statistics about the evolutionary process.

## Properties

| Property | Summary |
|:-----|:--------|
| `AverageFitness` | Gets or sets the average fitness of the current population. |
| `BestFitness` | Gets or sets the best fitness found so far. |
| `BestIndividual` | Gets or sets a reference to the best individual found so far. |
| `Diversity` | Gets or sets a measure of the population's genetic diversity. |
| `FitnessHistory` | Gets or sets the fitness history across generations. |
| `FitnessStandardDeviation` | Gets or sets the standard deviation of fitness in the population. |
| `Generation` | Gets or sets the current generation number. |
| `GenerationsSinceImprovement` | Gets or sets the number of generations since the last improvement. |
| `ImprovedInLastGeneration` | Gets or sets whether a fitness improvement occurred in the last generation. |
| `TimeElapsed` | Gets or sets the time elapsed since the evolution started. |
| `WorstFitness` | Gets or sets the worst fitness in the current population. |

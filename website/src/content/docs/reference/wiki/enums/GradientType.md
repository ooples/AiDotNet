`Enums` · `AiDotNet.Enums`

Specifies different types of gradient descent optimization algorithms used in machine learning.

## For Beginners

Gradient descent is how AI models learn from data. Think of it like finding the 
lowest point in a valley by taking steps downhill. Different gradient types represent different 
strategies for how to take these steps - some are faster but riskier, others are slower but more reliable.

## Fields

| Field | Summary |
|:-----|:--------|
| `Alpha` | Represents a gradient descent algorithm that prioritizes speed of convergence. |
| `Beta` | Represents a gradient descent algorithm that prioritizes stability and reliability over speed. |
| `Omega` | Represents the standard gradient descent algorithm with momentum and adaptive learning rates. |

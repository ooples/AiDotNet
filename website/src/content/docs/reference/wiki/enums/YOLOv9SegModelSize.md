## For Beginners

YOLOv9-Seg extends YOLOv9 with instance segmentation using Programmable
Gradient Information (PGI) and Generalized ELAN (GELAN) architecture.

## How It Works

**Reference:** Wang et al., "YOLOv9: Learning What You Want to Learn Using Programmable
Gradient Information", arXiv 2024.

Defines the type of infrastructure component (Tier 3 metadata).

## For Beginners

Infrastructure components handle storage, serving, and platform
concerns. They don't contain ML logic — they support the ML system.

## Fields

| Field | Summary |
|:-----|:--------|
| `Cache` | Caching layer for model predictions or intermediate results. |
| `Configuration` | Configuration management infrastructure. |
| `Metrics` | Metrics collection and monitoring infrastructure. |
| `Persistence` | Model serialization and persistence infrastructure. |

| `ServingEndpoint` | Model serving endpoint for production inference. |

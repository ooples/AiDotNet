Statistics about regression residuals.

## Properties

| Property | Summary |
|:-----|:--------|
| `DurbinWatson` | Durbin-Watson statistic (2 for no autocorrelation). |
| `Kurtosis` | Kurtosis of residuals (3 for normal). |
| `Max` | Maximum residual. |
| `Mean` | Mean of residuals (should be close to 0). |
| `Median` | Median residual. |
| `Min` | Minimum residual. |
| `OutlierCount` | Number of outlier residuals. |
| `Skewness` | Skewness of residuals (0 for normal). |
| `StdDev` | Standard deviation of residuals. |

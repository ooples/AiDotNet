`Enums` · `AiDotNet.Evaluation.Enums`

Specifies the output format for evaluation reports.

## For Beginners

After evaluating your model, you'll want to share or save the results.
Different formats serve different purposes:

- **Console:** Quick viewing during development
- **Markdown:** Documentation and reports
- **JSON:** Integration with other tools and programmatic access
- **CSV:** Spreadsheet analysis and comparison

## How It Works

Evaluation results can be exported in various formats depending on the use case:
documentation, integration with other tools, or human readability.

## Fields

| Field | Summary |
|:-----|:--------|
| `Console` | Plain text format for console output with ASCII formatting. |
| `Csv` | CSV format for spreadsheet import and data analysis. |
| `Html` | HTML format for web display and interactive reports. |

| `JupyterNotebook` | Jupyter notebook cell output format with rich display. |

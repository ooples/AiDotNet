`Options & Configuration` · `AiDotNet.TextToSpeech.Vocoders`

Options for iSTFTNet (inverse STFT-based vocoder that outputs STFT coefficients then iSTFT).

## For Beginners

These options configure the ISTFTNet model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `ISTFTNetOptions(ISTFTNetOptions)` | Initializes a new instance by copying from another instance. |

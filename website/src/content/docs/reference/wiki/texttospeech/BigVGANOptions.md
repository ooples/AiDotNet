`Options & Configuration` · `AiDotNet.TextToSpeech.Vocoders`

Options for BigVGAN (universal vocoder with anti-aliased multi-periodicity composition and Snake activation).

## For Beginners

These options configure the BigVGAN model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `BigVGANOptions(BigVGANOptions)` | Initializes a new instance by copying from another instance. |

`Options & Configuration` · `AiDotNet.TextToSpeech.Classic`

Options for Tacotron 2 (location-sensitive attention with WaveNet vocoder).

## For Beginners

These options configure the Tacotron2 model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `Tacotron2Options(Tacotron2Options)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `AttentionLocationChannels` | Gets or sets the attention location feature channels. |
| `AttentionRnnDim` | Gets or sets the attention RNN dimension. |
| `DecoderRnnDim` | Gets or sets the decoder RNN dimension. |
| `PrenetDim` | Gets or sets the prenet dimension. |

`Options & Configuration` · `AiDotNet.TextToSpeech.ProprietaryAPI`

Options for GoogleCloudTTS TTS API wrapper.

## For Beginners

These options configure the GoogleCloudTTS model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `GoogleCloudTTSOptions(GoogleCloudTTSOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `ApiEndpoint` | API endpoint URL for the Google Cloud TTS service. |
| `ApiKey` | API key for the Google Cloud TTS service. |
| `VoiceId` | Voice identifier to use for synthesis. |

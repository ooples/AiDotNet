`Models & Types` · `AiDotNet.TextToSpeech.ProprietaryAPI`

Amazon Polly: AWS neural TTS service with neural and standard engines.

## For Beginners

Amazon Polly converts text into lifelike speech using deep learning.
It offers multiple voices and languages, with neural and standard synthesis engines.
The neural engine produces more natural-sounding speech but requires more compute.
This local implementation provides API-compatible inference for offline use.

## Methods

| Method | Summary |
|:-----|:--------|
| `Synthesize(String)` | Synthesizes speech using AmazonPolly's API-compatible local inference pipeline. |

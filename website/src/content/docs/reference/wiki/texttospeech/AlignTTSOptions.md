`Options & Configuration` · `AiDotNet.TextToSpeech.Classic`

Options for AlignTTS (alignment-free non-autoregressive TTS with mix density network).

## For Beginners

These options configure the AlignTTS model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `AlignTTSOptions(AlignTTSOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `NumMixtures` | Gets or sets the number of mixture components for the alignment loss. |

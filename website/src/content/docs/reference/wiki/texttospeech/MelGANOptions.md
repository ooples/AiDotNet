`Options & Configuration` · `AiDotNet.TextToSpeech.Vocoders`

Options for MelGAN (lightweight GAN vocoder with no need for paired training data).

## For Beginners

These options configure the MelGAN model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `MelGANOptions(MelGANOptions)` | Initializes a new instance by copying from another instance. |

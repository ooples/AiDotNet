`Options & Configuration` · `AiDotNet.TextToSpeech.Classic`

Options for Glow-TTS (flow-based non-autoregressive TTS with monotonic alignment search).

## For Beginners

These options configure the GlowTTS model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `GlowTTSOptions(GlowTTSOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `NumFlowLayers` | Gets or sets the number of flow coupling layers. |
| `Temperature` | Gets or sets the temperature for sampling during inference. |

`Options & Configuration` · `AiDotNet.TextToSpeech.Vocoders`

Options for HiFi-GAN (high-fidelity GAN-based vocoder with multi-receptive field fusion).

## For Beginners

These options configure the HiFiGAN model. Default values follow the original paper settings.

## Properties

| Property | Summary |
|:-----|:--------|
| `ResblockDilationSizes` | Gets or sets the resblock dilation sizes for each kernel. |

`Options & Configuration` · `AiDotNet.TextToSpeech.ProprietaryAPI`

Options for PlayHT TTS API wrapper.

## For Beginners

These options configure the PlayHT model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `PlayHTOptions(PlayHTOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `ApiEndpoint` | API endpoint URL for the PlayHT TTS service. |
| `ApiKey` | API key for the PlayHT TTS service. |
| `VoiceId` | Voice identifier to use for synthesis. |

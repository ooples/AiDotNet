`Options & Configuration` · `AiDotNet.TextToSpeech.Classic`

Options for AdaSpeech (adaptive TTS with acoustic condition modeling).

## For Beginners

These options configure the AdaSpeech model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `AdaSpeechOptions(AdaSpeechOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `ConditionDim` | Gets or sets the acoustic condition embedding dimension. |
| `NumConditionLayers` | Gets or sets the number of conditional layer normalization layers. |

`Options & Configuration` · `AiDotNet.TextToSpeech.Classic`

Options for Tacotron (attention-based seq2seq TTS).

## For Beginners

These options configure the Tacotron model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `TacotronOptions(TacotronOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `CbhgBankSize` | Gets or sets the CBHG encoder bank size. |
| `PrenetDim` | Gets or sets the prenet dim for decoder. |

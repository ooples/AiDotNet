`Models & Types` · `AiDotNet.TextToSpeech.ProprietaryAPI`

ElevenLabs: AI voice synthesis platform with voice cloning, emotion control, and multilingual support.

## For Beginners

ElevenLabs provides highly realistic AI voice synthesis with
voice cloning (create a custom voice from a few minutes of audio), emotion control,
and multilingual support across 29+ languages. This local implementation provides
offline inference compatible with the ElevenLabs API surface.

## Methods

| Method | Summary |
|:-----|:--------|
| `Synthesize(String)` | Synthesizes speech using ElevenLabs's API-compatible local inference pipeline. |

`Models & Types` · `AiDotNet.TextToSpeech.ProprietaryAPI`

Azure Neural TTS: local neural synthesis model inspired by Microsoft's architecture.

## For Beginners

Azure Neural TTS uses Microsoft's neural speech synthesis architecture
to generate natural-sounding speech from text. It supports expressive styles like newscast,
cheerful, and customer service tones. This local implementation provides offline inference
compatible with the Azure Cognitive Services API surface.

## Methods

| Method | Summary |
|:-----|:--------|
| `Synthesize(String)` | Synthesizes speech using a local neural synthesis pipeline. |

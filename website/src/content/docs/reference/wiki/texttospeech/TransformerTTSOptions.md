`Options & Configuration` · `AiDotNet.TextToSpeech.Classic`

Options for Transformer TTS (multi-head self-attention acoustic model).

## For Beginners

These options configure the TransformerTTS model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `TransformerTTSOptions(TransformerTTSOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `FeedForwardDim` | Gets or sets the feedforward dimension in transformer blocks. |
| `PositionalEncodingDim` | Gets or sets the positional encoding dimension. |

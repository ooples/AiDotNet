Options for Parallel WaveGAN (non-autoregressive GAN vocoder with multi-resolution STFT loss).

## For Beginners

These options configure the ParallelWaveGAN model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `ParallelWaveGANOptions(ParallelWaveGANOptions)` | Initializes a new instance by copying from another instance. |

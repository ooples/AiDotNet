`Models & Types` · `AiDotNet.DataVersioning`

Represents differences between two data versions.

## Properties

| Property | Summary |
|:-----|:--------|
| `FilesAdded` | Gets or sets files added in version 2. |
| `FilesModified` | Gets or sets files modified between versions. |
| `FilesRemoved` | Gets or sets files removed in version 2. |
| `FilesUnchanged` | Gets or sets unchanged files. |
| `SizeDelta` | Gets the size change in bytes. |
| `Summary` | Gets a summary of changes. |
| `Version1` | Gets or sets the first version. |
| `Version2` | Gets or sets the second version. |

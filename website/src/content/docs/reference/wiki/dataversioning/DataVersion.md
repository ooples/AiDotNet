`Models & Types` · `AiDotNet.DataVersioning`

Represents a specific version of a dataset.

## Properties

| Property | Summary |
|:-----|:--------|
| `ContentHash` | Gets or sets the content hash (SHA-256). |
| `CreatedAt` | Gets or sets when this version was created. |
| `DataPath` | Gets or sets the path to the versioned data. |
| `DatasetId` | Gets or sets the dataset this version belongs to. |
| `FileCount` | Gets or sets the number of files. |
| `Files` | Gets or sets file-level information. |
| `Message` | Gets or sets the version message. |
| `Metadata` | Gets or sets additional metadata. |
| `SizeBytes` | Gets or sets the total size in bytes. |
| `SizeFormatted` | Gets the size in a human-readable format. |
| `VersionId` | Gets or sets the version identifier (content hash). |
| `VersionNumber` | Gets or sets the version number (sequential). |

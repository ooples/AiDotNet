`Models & Types` · `AiDotNet.DataVersioning`

Represents a dataset registered for version control.

## Properties

| Property | Summary |
|:-----|:--------|
| `CreatedAt` | Gets or sets when the dataset was created. |
| `DatasetId` | Gets or sets the unique dataset identifier. |
| `Description` | Gets or sets the dataset description. |
| `LastUpdatedAt` | Gets or sets when the dataset was last updated. |
| `LatestVersionId` | Gets or sets the latest version ID. |
| `Metadata` | Gets or sets additional metadata. |
| `Name` | Gets or sets the dataset name. |
| `VersionCount` | Gets or sets the number of versions. |

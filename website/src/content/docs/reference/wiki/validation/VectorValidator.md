`Helpers & Utilities` · `AiDotNet.Validation`

Provides validation methods for vectors and related operations.

## Methods

| Method | Summary |
|:-----|:--------|
| `ValidateLength(Vector<>,Int32,String,String)` | Validates that the vector length matches the expected length. |
| `ValidateLengthForShape(Vector<>,Int32[],String,String)` | Validates that the vector length matches the expected size based on the shape. |

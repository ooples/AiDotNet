`Helpers & Utilities` · `AiDotNet.Validation`

Provides validation methods for serialization and deserialization operations.

## Methods

| Method | Summary |
|:-----|:--------|
| `ValidateFilePath(String,String,String)` | Validates that a file path is not null or empty. |
| `ValidateLayerTypeExists(String,Type,String,String)` | Validates that a layer type exists and can be instantiated. |
| `ValidateLayerTypeName(String,String,String)` | Validates that a layer type name is not null or empty. |
| `ValidateReader(BinaryReader,String,String)` | Validates that a binary reader is not null. |
| `ValidateStream(Stream,Boolean,Boolean,String,String)` | Validates that a stream is not null and has the expected capabilities. |
| `ValidateVersion(Int32,Int32,String,String)` | Validates that a serialized version matches the expected version. |
| `ValidateWriter(BinaryWriter,String,String)` | Validates that a binary writer is not null. |

`Helpers & Utilities` · `AiDotNet.Validation`

Provides validation methods for regression operations.

## Methods

| Method | Summary |
|:-----|:--------|
| `ValidateDataValues(Matrix<>,Vector<>,String,String)` | Validates that the input data doesn't contain any NaN or infinity values. |
| `ValidateFeatureCount(Matrix<>,Int32,String,String)` | Validates that the input matrix has the expected number of feature columns. |
| `ValidateInputOutputDimensions(Matrix<>,Vector<>,String,String)` | Validates that the input and output data have compatible dimensions for regression. |

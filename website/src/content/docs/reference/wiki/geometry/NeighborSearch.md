`Helpers & Utilities` · `AiDotNet.Geometry.Preprocessing`

Provides neighbor search utilities for point cloud data.

## Methods

| Method | Summary |
|:-----|:--------|
| `BallQuery(PointCloudData<>,PointCloudData<>,Double,Int32)` | Performs ball query for PointNet++ style grouping. |
| `ComputeDistanceMatrix(PointCloudData<>)` | Computes a distance matrix between all pairs of points. |
| `KNearestNeighbors(PointCloudData<>,Int32)` | Finds the k nearest neighbors for each point in the cloud. |
| `RadiusSearch(PointCloudData<>,PointCloudData<>,Double)` | Finds all points within a given radius of each query point. |

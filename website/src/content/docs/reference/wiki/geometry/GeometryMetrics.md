`Helpers & Utilities` · `AiDotNet.Geometry.Preprocessing`

Provides quality metrics for comparing 3D geometry (point clouds, meshes).

## Methods

| Method | Summary |
|:-----|:--------|
| `ChamferDistance(PointCloudData<>,PointCloudData<>,Boolean)` | Computes the Chamfer Distance between two point clouds. |
| `FScore(PointCloudData<>,PointCloudData<>,Double)` | Computes the F-Score between two point clouds at a given threshold. |
| `HausdorffDistance(PointCloudData<>,PointCloudData<>)` | Computes the Hausdorff Distance between two point clouds. |

`Helpers & Utilities` · `AiDotNet.Geometry.IO`

OBJ mesh and point cloud IO utilities.

## Methods

| Method | Summary |
|:-----|:--------|
| `LoadMesh(String,Boolean)` | Loads a triangle mesh from an OBJ file. |
| `LoadPointCloud(String)` | Loads a point cloud from an OBJ file (using vertex lines). |
| `SaveMesh(TriangleMeshData<>,String,Boolean,Boolean,Boolean)` | Saves a triangle mesh to an OBJ file. |
| `SavePointCloud(PointCloudData<>,String,Boolean)` | Saves a point cloud to an OBJ file (vertex lines only). |

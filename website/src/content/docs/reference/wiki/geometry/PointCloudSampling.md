`Helpers & Utilities` · `AiDotNet.Geometry.Preprocessing`

Provides sampling utilities for point cloud data.

## Methods

| Method | Summary |
|:-----|:--------|
| `FarthestPointSample(PointCloudData<>,Int32,Nullable<Int32>)` | Performs farthest point sampling (FPS) on a point cloud. |
| `PoissonDiskSample(PointCloudData<>,Double,Int32,Nullable<Int32>)` | Performs Poisson disk sampling on a point cloud. |
| `UniformSample(PointCloudData<>,Int32,Nullable<Int32>)` | Performs uniform random sampling on a point cloud. |
| `VoxelGridSample(PointCloudData<>,Double)` | Samples points by voxel grid downsampling. |

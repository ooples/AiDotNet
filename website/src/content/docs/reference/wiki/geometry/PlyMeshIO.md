`Helpers & Utilities` · `AiDotNet.Geometry.IO`

PLY mesh and point cloud IO utilities.

## Methods

| Method | Summary |
|:-----|:--------|
| `LoadMesh(String,Boolean)` | Loads a triangle mesh from a PLY file. |
| `LoadPointCloud(String)` | Loads a point cloud from a PLY file (vertex element only). |
| `SaveMesh(TriangleMeshData<>,String,Boolean)` | Saves a triangle mesh to a PLY file. |
| `SavePointCloud(PointCloudData<>,String,Boolean,Boolean,Boolean,Boolean)` | Saves a point cloud to a PLY file. |

Configuration options for Bagging classifier.

## Properties

| Property | Summary |
|:-----|:--------|
| `Bootstrap` | Gets or sets whether to sample with replacement. |
| `MaxFeatures` | Gets or sets the fraction of features to use for each estimator. |
| `MaxSamples` | Gets or sets the fraction of samples to draw for each estimator. |
| `NumEstimators` | Gets or sets the number of base estimators. |

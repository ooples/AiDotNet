`Options & Configuration` · `AiDotNet.Autodiff.Testing`

Configuration for gradient verification.

## Properties

| Property | Summary |
|:-----|:--------|
| `AbsoluteTolerance` | Absolute tolerance for gradient comparison (default: 1e-6). |
| `Epsilon` | Step size for finite differences (default: 1e-5). |
| `RandomSeed` | Random seed for reproducible test data (default: 42). |
| `RelativeTolerance` | Relative tolerance for gradient comparison (default: 1e-4). |
| `Verbose` | Whether to print detailed results (default: false). |

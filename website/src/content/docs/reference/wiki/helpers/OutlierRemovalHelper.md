`Helpers & Utilities` · `AiDotNet.Helpers`

Provides helper methods for outlier removal algorithms.

## For Beginners

This helper class provides common functionality used by different
outlier removal algorithms. It handles converting between different data types and ensures
that the algorithms can work with various input and output formats.

## Methods

| Method | Summary |
|:-----|:--------|
| `ConvertToMatrixVector(,)` | Converts generic input and output types to concrete Matrix and Vector types. |
| `ConvertToOriginalTypes(Matrix<>,Vector<>,Type,Type)` | Converts Matrix and Vector data back to the original generic types. |

`Helpers & Utilities` · `AiDotNet.Helpers`

Provides text processing utilities for splitting and tokenizing text.

## Methods

| Method | Summary |
|:-----|:--------|
| `SplitIntoSentences(String)` | Splits text into sentences based on common sentence-ending punctuation. |
| `Tokenize(String)` | Tokenizes text by splitting on whitespace and common punctuation marks. |

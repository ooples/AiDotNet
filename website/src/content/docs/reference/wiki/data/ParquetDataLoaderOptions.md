`Options & Configuration` · `AiDotNet.Data.Formats`

Configuration options for the `ParquetDataLoader`.

## Properties

| Property | Summary |
|:-----|:--------|
| `FeatureColumns` | Column names to use as features. |
| `FilePath` | Path to the Parquet file. |
| `LabelColumn` | Column name to use as the label. |
| `MaxRows` | Optional maximum number of rows to load. |
| `RandomSeed` | Optional random seed for reproducible sampling when MaxRows is set. |

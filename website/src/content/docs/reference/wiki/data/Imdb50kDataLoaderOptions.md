`Options & Configuration` · `AiDotNet.Data.Text.Benchmarks`

Configuration options for the IMDB 50k sentiment analysis data loader.

## Properties

| Property | Summary |
|:-----|:--------|
| `AutoDownload` | Automatically download if not present. |
| `DataPath` | Root data path. |
| `MaxSamples` | Optional maximum number of samples to load. |
| `MaxSequenceLength` | Maximum number of tokens (words) per review. |
| `Split` | Dataset split to load. |
| `VocabularySize` | Maximum vocabulary size (most frequent words). |

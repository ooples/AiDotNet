`Options & Configuration` · `AiDotNet.Data.Vision.Benchmarks`

Configuration options for the Oxford Flowers-102 dataset (Nilsback & Zisserman 2008).

## How It Works

Flowers-102 — 102 fine-grained flower species, 8,189 images total.
Standard fine-grained classification benchmark. The `setid.mat`
file defines the canonical 1,020/1,020/6,149 train/val/test split.

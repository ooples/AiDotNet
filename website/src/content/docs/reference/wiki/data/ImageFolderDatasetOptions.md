`Options & Configuration` · `AiDotNet.Data.Vision`

Configuration options for the `ImageFolderDataset`.

## Properties

| Property | Summary |
|:-----|:--------|
| `Channels` | Number of color channels. |
| `Extensions` | File extensions to include. |
| `ImageHeight` | Target image height in pixels. |
| `ImageWidth` | Target image width in pixels. |
| `MaxSamples` | Optional maximum number of samples to load. |
| `NormalizePixels` | Whether to normalize pixel values to [0, 1]. |
| `RandomSeed` | Optional random seed for reproducible sampling. |
| `Recursive` | Whether to search subdirectories recursively. |
| `RootDirectory` | Root directory containing class subdirectories. |
| `Split` | Dataset split to load. |

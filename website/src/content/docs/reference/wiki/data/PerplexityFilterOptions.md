`Options & Configuration` · `AiDotNet.Data.Quality`

Configuration options for perplexity-based text quality filtering.

## Properties

| Property | Summary |
|:-----|:--------|
| `MaxPerplexity` | Maximum perplexity threshold. |
| `MinPerplexity` | Minimum perplexity threshold. |
| `NGramOrder` | N-gram order for the language model. |
| `SmoothingFactor` | Smoothing factor for unseen n-grams. |

## Methods

| Method | Summary |
|:-----|:--------|
| `Validate` | Validates that all option values are within acceptable ranges. |

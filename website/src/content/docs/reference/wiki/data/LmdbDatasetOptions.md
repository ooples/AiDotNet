`Options & Configuration` · `AiDotNet.Data.Formats`

Configuration options for LMDB-based dataset access.

## Properties

| Property | Summary |
|:-----|:--------|
| `DataPath` | Path to the LMDB environment directory. |
| `MapSize` | Maximum size of the LMDB map in bytes. |
| `MaxReaders` | Maximum number of readers. |
| `ReadOnly` | Whether to open the database in read-only mode. |

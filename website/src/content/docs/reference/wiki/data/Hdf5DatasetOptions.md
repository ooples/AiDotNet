`Options & Configuration` · `AiDotNet.Data.Formats`

Configuration options for HDF5 dataset access.

## Properties

| Property | Summary |
|:-----|:--------|
| `ChunkSize` | Number of samples to read per chunk for buffered I/O. |
| `FeaturesDataset` | Name of the features dataset within the file. |
| `FilePath` | Path to the HDF5 file. |
| `LabelsDataset` | Name of the labels dataset within the file. |

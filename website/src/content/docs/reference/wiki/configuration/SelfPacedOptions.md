Options specific to self-paced curriculum learning.

## Properties

| Property | Summary |
|:-----|:--------|
| `InitialLambda` | Gets or sets the initial pace threshold (lambda). |
| `LambdaGrowthRate` | Gets or sets how much to increase lambda each epoch. |
| `MaxLambda` | Gets or sets the maximum pace threshold. |
| `Regularizer` | Gets or sets the self-pace regularizer type. |

`Options & Configuration` · `AiDotNet.Configuration`

Configuration options for federated tabular benchmark suites.

## For Beginners

Tabular benchmarks test models on spreadsheet-like data (rows and columns).

## How It Works

This groups dataset-specific options for tabular benchmarks under a single facade-facing configuration object.

## Properties

| Property | Summary |
|:-----|:--------|
| `NonIid` | Gets or sets options for the synthetic non-IID tabular suite. |

`Options & Configuration` · `AiDotNet.Configuration`

Configuration for checkpointing during RL training.

## Properties

| Property | Summary |
|:-----|:--------|
| `CheckpointDirectory` | Directory to save checkpoints. |
| `KeepBestN` | Keep only the best N checkpoints. |
| `SaveEveryEpisodes` | Save checkpoint every N episodes. |
| `SaveOnBestReward` | Save checkpoint on best reward improvement. |

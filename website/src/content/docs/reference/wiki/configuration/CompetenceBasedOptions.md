`Options & Configuration` · `AiDotNet.Configuration`

Options specific to competence-based curriculum learning.

## Properties

| Property | Summary |
|:-----|:--------|
| `CompetenceThreshold` | Gets or sets the competence threshold required to advance phases. |
| `MetricType` | Gets or sets the type of competence metric to use. |
| `MinImprovement` | Gets or sets the minimum improvement to reset patience. |
| `PatienceEpochs` | Gets or sets the patience epochs for plateau detection. |
| `SmoothingFactor` | Gets or sets the exponential smoothing factor for competence updates. |

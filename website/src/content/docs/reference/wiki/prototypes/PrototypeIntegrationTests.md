`Helpers & Utilities` · `AiDotNet.Prototypes`

Integration tests for Phase A prototype validation.
Tests GPU vs CPU performance, multi-type support, and numerical accuracy.

## Methods

| Method | Summary |
|:-----|:--------|
| `BenchmarkGpuVsCpu` | Benchmarks GPU vs CPU performance for vector operations. |
| `RunAll` | Runs all Phase A integration tests. |
| `TestAdamOptimizer` | Tests Adam optimizer convergence. |
| `TestLinearRegression` | Tests linear regression on synthetic data. |
| `TestVectorOperations` | Tests basic vector operations with different data types. |

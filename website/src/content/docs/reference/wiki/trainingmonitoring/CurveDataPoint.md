`Models & Types` · `AiDotNet.TrainingMonitoring.Dashboard`

Represents a PR/ROC curve data point.

## Properties

| Property | Summary |
|:-----|:--------|
| `AUC` | Gets or sets the area under the curve. |
| `Step` | Gets or sets the training step. |
| `Thresholds` | Gets or sets the thresholds. |
| `WallTime` | Gets or sets the wall clock time. |
| `XValues` | Gets or sets the X values (e.g., recall for PR, FPR for ROC). |
| `YValues` | Gets or sets the Y values (e.g., precision for PR, TPR for ROC). |

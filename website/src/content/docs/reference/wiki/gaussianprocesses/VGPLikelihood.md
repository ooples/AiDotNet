Specifies the likelihood function type for Variational Gaussian Process.

## For Beginners

The likelihood function describes what type of observations
you have and how they relate to the underlying latent function.

## Fields

| Field | Summary |
|:-----|:--------|
| `Bernoulli` | Bernoulli likelihood for binary classification. |
| `Gaussian` | Gaussian likelihood for continuous regression with normally distributed noise. |
| `Poisson` | Poisson likelihood for count data. |

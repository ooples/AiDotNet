`Models & Types` · `AiDotNet.Interfaces`

Metadata about a single segmented medical structure.

## Properties

| Property | Summary |
|:-----|:--------|
| `ClassId` | Class ID of this structure. |
| `MeanConfidence` | Mean confidence score across all voxels/pixels in this structure. |
| `Name` | Human-readable name of this structure (e.g., "Left Kidney", "Liver"). |
| `VolumeOrArea` | Volume of the structure in voxels (for 3D) or area in pixels (for 2D). |

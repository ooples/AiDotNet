`Options & Configuration` · `AiDotNet.Interfaces`

Configuration for microphone array geometry.

## Properties

| Property | Summary |
|:-----|:--------|
| `NumMicrophones` | Gets or sets the number of microphones. |
| `Positions` | Gets or sets microphone positions [num_mics, 3] (x, y, z). |
| `Spacing` | Gets or sets the spacing between microphones (for regular arrays). |
| `Type` | Gets or sets the array type. |

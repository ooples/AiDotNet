`Interfaces` · `AiDotNet.Interfaces`

Extended dataset interface with additional metadata and features.

## Properties

| Property | Summary |
|:-----|:--------|
| `ClassCount` | Gets the number of classes for classification problems. |
| `ClassLabels` | Gets the unique class labels for classification problems. |
| `FeatureCount` | Gets the number of features per input sample. |
| `FeatureNames` | Gets the feature names if available. |
| `IsClassification` | Gets whether this is a classification or regression dataset. |
| `Metadata` | Gets metadata for the dataset. |
| `SampleWeights` | Gets the sample weights if available. |

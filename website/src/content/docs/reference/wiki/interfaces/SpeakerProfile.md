`Models & Types` · `AiDotNet.Interfaces`

Represents an enrolled speaker profile.

## Properties

| Property | Summary |
|:-----|:--------|
| `CreatedAt` | Gets or sets when this profile was created. |
| `Embedding` | Gets or sets the speaker embedding. |
| `NumEnrollmentSamples` | Gets or sets the number of audio samples used to create this profile. |
| `SpeakerId` | Gets or sets the unique identifier for this speaker. |
| `TotalEnrollmentDuration` | Gets or sets the total duration of enrollment audio in seconds. |
| `UpdatedAt` | Gets or sets when this profile was last updated. |

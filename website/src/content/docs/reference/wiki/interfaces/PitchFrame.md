`Models & Types` · `AiDotNet.Interfaces`

Represents a single pitch detection frame.

## Properties

| Property | Summary |
|:-----|:--------|
| `Confidence` | Gets the detection confidence (0-1). |
| `IsVoiced` | Gets whether this frame is voiced (contains pitched content). |
| `Pitch` | Gets the detected pitch in Hz. |
| `Time` | Gets the timestamp in seconds. |

`Models & Types` · `AiDotNet.Interfaces`

Represents a speaker segment in diarization output.

## Properties

| Property | Summary |
|:-----|:--------|
| `Confidence` | Gets or sets the confidence score for this segment. |
| `Duration` | Gets the duration of this segment in seconds. |
| `EndTime` | Gets or sets the end time in seconds. |
| `Speaker` | Gets or sets the speaker label (e.g., "Speaker_0" or actual name if known). |
| `StartTime` | Gets or sets the start time in seconds. |

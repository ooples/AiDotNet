`Options & Configuration` · `AiDotNet.Interfaces`

Options for audio feature extraction.

## Properties

| Property | Summary |
|:-----|:--------|
| `CenterPad` | Gets or sets whether to center-pad the signal. |
| `EffectiveWindowLength` | Gets the effective window length. |
| `FftSize` | Gets or sets the FFT window size in samples. |
| `HopLength` | Gets or sets the hop length (stride) between frames in samples. |
| `SampleRate` | Gets or sets the sample rate of the audio. |
| `WindowLength` | Gets or sets the window length in samples. |

`Models & Types` · `AiDotNet.Interfaces`

A segment of audio with a detected chord.

## Properties

| Property | Summary |
|:-----|:--------|
| `Chord` | Gets or sets the chord symbol (e.g., "C", "Am", "G7"). |
| `Confidence` | Gets or sets the confidence score for this chord. |
| `Duration` | Gets the duration of this segment. |
| `EndTime` | Gets or sets the end time in seconds. |
| `Quality` | Gets or sets the chord quality (e.g., "major", "minor", "dominant7"). |
| `Root` | Gets or sets the root note of the chord. |
| `StartTime` | Gets or sets the start time in seconds. |

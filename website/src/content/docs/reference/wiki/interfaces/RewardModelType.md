Types of reward models.

## For Beginners

This enum distinguishes between the two main types of reward models.

## Fields

| Field | Summary |
|:-----|:--------|
| `Hybrid` | Hybrid - combines both process and outcome scoring. |
| `Outcome` | Outcome Reward Model - scores only the final answer. |
| `Process` | Process Reward Model - scores individual reasoning steps. |

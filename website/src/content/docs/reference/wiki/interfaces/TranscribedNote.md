`Models & Types` · `AiDotNet.Interfaces`

Represents a single transcribed musical note.

## Properties

| Property | Summary |
|:-----|:--------|
| `Confidence` | Gets the confidence/velocity of the note (0-1). |
| `Duration` | Gets the duration of the note in seconds. |
| `EndTime` | Gets the end time in seconds. |
| `MidiNote` | Gets the MIDI note number (21-108 for piano). |
| `NoteName` | Gets the note name (e.g., "C4", "A#3"). |
| `PitchHz` | Gets the pitch in Hz. |
| `StartTime` | Gets the start time in seconds. |

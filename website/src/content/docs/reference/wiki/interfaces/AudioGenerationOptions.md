`Options & Configuration` · `AiDotNet.Interfaces`

Advanced options for audio generation.

## Properties

| Property | Summary |
|:-----|:--------|
| `DurationSeconds` | Gets or sets the duration of audio to generate in seconds. |
| `GuidanceScale` | Gets or sets the guidance scale. |
| `NumInferenceSteps` | Gets or sets the number of inference steps. |
| `SchedulerType` | Gets or sets the scheduler type for the diffusion process. |
| `Seed` | Gets or sets the random seed. |
| `Stereo` | Gets or sets whether to generate stereo audio. |

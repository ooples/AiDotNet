`Models & Types` · `AiDotNet.Interfaces`

Represents an adjustable parameter for an audio effect.

## Properties

| Property | Summary |
|:-----|:--------|
| `CurrentValue` | Gets or sets the current value. |
| `DefaultValue` | Gets the default value. |
| `Description` | Gets the description of what this parameter does. |
| `DisplayName` | Gets the display name for UI. |
| `MaxValue` | Gets the maximum allowed value. |
| `MinValue` | Gets the minimum allowed value. |
| `Name` | Gets the parameter name. |
| `Unit` | Gets the unit of measurement (e.g., "dB", "Hz", "ms", "%"). |

Information about an available TTS voice.

## Properties

| Property | Summary |
|:-----|:--------|
| `Gender` | Gets or sets the gender of this voice. |
| `Id` | Gets or sets the unique identifier for this voice. |
| `Language` | Gets or sets the language code for this voice. |
| `Name` | Gets or sets the display name of this voice. |
| `SpeakerEmbedding` | Gets or sets the pre-computed speaker embedding for this voice. |
| `Style` | Gets or sets the style description of this voice. |

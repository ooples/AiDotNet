`Models & Types` · `AiDotNet.Interfaces`

Represents an audio fingerprint.

## Properties

| Property | Summary |
|:-----|:--------|
| `Algorithm` | Gets or sets the algorithm used to create the fingerprint. |
| `Data` | Gets or sets the fingerprint data. |
| `Duration` | Gets or sets the duration of the fingerprinted audio in seconds. |
| `FrameCount` | Gets or sets the number of frames in the fingerprint. |
| `Hash` | Gets or sets the binary hash (for compact storage/comparison). |
| `Metadata` | Gets or sets optional metadata about the fingerprint. |
| `SampleRate` | Gets or sets the sample rate of the source audio. |

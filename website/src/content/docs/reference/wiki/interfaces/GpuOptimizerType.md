| Field | Summary |
|:-----|:--------|

| `Adam` | Adam optimizer with adaptive learning rates. |
| `AdamW` | AdamW optimizer with decoupled weight decay. |
| `Lamb` | Layer-wise Adaptive Moments optimizer. |
| `Lars` | Layer-wise Adaptive Rate Scaling. |
| `Nag` | Nesterov Accelerated Gradient with lookahead. |
| `RmsProp` | RMSprop optimizer with moving average of squared gradients. |
| `Sgd` | Simple SGD with optional momentum. |

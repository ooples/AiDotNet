Represents the result of emotion recognition.

## Properties

| Property | Summary |
|:-----|:--------|
| `Arousal` | Gets the arousal level (-1.0 to 1.0). |
| `Confidence` | Gets the confidence score (0.0 to 1.0). |
| `Emotion` | Gets the primary detected emotion. |
| `SecondaryEmotion` | Gets the secondary emotion (if detected with significant confidence). |
| `Valence` | Gets the valence level (-1.0 to 1.0). |

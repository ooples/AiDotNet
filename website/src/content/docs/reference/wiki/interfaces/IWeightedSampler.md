Interface for samplers that use sample weights.

## Properties

| Property | Summary |
|:-----|:--------|
| `NumSamples` | Gets or sets the number of samples to draw per epoch. |
| `Replacement` | Gets or sets whether to sample with replacement. |
| `Weights` | Gets or sets the weights for each sample. |

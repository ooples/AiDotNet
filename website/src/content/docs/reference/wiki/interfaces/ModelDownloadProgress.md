`Models & Types` · `AiDotNet.Interfaces`

Progress information for model downloads.

## Properties

| Property | Summary |
|:-----|:--------|
| `BytesDownloaded` | Gets or sets the number of bytes downloaded for the current file. |
| `CurrentFile` | Gets or sets the current file being downloaded. |
| `FileProgress` | Gets or sets the current file progress (0.0 to 1.0). |
| `OverallProgress` | Gets or sets the overall progress (0.0 to 1.0). |
| `TotalBytes` | Gets or sets the total bytes for the current file (-1 if unknown). |

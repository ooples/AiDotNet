`Interfaces` · `AiDotNet.Interfaces`

Metadata about an ONNX model including inputs, outputs, and opset version.

## Properties

| Property | Summary |
|:-----|:--------|
| `Description` | Gets the model description if available. |
| `Inputs` | Gets information about all input tensors. |
| `ModelName` | Gets the model name from the ONNX file. |
| `OpsetVersion` | Gets the ONNX opset version used by this model. |
| `Outputs` | Gets information about all output tensors. |
| `ProducerName` | Gets the producer name (e.g., "pytorch", "tensorflow"). |
| `ProducerVersion` | Gets the producer version. |

`Models & Types` · `AiDotNet.Interfaces`

Represents the result of language identification.

## Properties

| Property | Summary |
|:-----|:--------|
| `AlternativeLanguage` | Gets the second most likely language (if close in probability). |
| `AlternativeProbability` | Gets the probability of the alternative language. |
| `Confidence` | Gets the confidence score (0.0 to 1.0). |
| `LanguageCode` | Gets the detected language code (ISO 639-1/3). |
| `LanguageName` | Gets the human-readable language name. |

Methods for interpolating between video frames.

## Fields

| Field | Summary |
|:-----|:--------|
| `Blend` | Blend-based interpolation (fast, reasonable quality). |
| `Diffusion` | Use diffusion-based interpolation (highest quality, slowest). |
| `Linear` | Simple linear interpolation (fast, lower quality). |
| `OpticalFlow` | Use optical flow-based interpolation (good quality, fast). |

Types of search algorithms available for Tree-of-Thoughts.

## For Beginners

Different ways to explore the tree of possibilities.
Each has different tradeoffs in terms of speed, memory, and quality.

## Fields

| Field | Summary |
|:-----|:--------|
| `BeamSearch` | Beam Search - keeps only the top N most promising paths at each level. |
| `BreadthFirst` | Breadth-First Search - explores all nodes at each level before going deeper. |

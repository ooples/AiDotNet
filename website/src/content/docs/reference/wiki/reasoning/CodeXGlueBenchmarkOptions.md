`Options & Configuration` · `AiDotNet.Reasoning.Benchmarks`

Options for configuring CodeXGLUE dataset loading.

## Properties

| Property | Summary |
|:-----|:--------|
| `CategoryField` | JSON field that contains an optional category label. |
| `DatasetFilePath` | Path to a JSONL dataset file. |
| `IdField` | JSON field that contains the record identifier. |
| `SourceField` | JSON field that contains the prompt/source. |
| `TargetField` | JSON field that contains the expected answer/target. |
| `TaskName` | Optional task label for display (e.g., "code-to-text", "text-to-code"). |

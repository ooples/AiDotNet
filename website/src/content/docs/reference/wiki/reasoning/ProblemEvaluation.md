`Models & Types` · `AiDotNet.Reasoning.Benchmarks.Models`

Result for a single problem evaluation.

## For Beginners

This records what happened when the system tried to solve
one specific problem.

## Properties

| Property | Summary |
|:-----|:--------|
| `Category` | Category of this problem. |
| `Confidence` | Confidence score for this answer. |
| `CorrectAnswer` | The correct answer. |
| `Duration` | Time taken to solve this problem. |
| `IsCorrect` | Whether the answer was correct. |
| `Metadata` | Additional metadata. |
| `Problem` | The problem statement. |
| `ProblemId` | The problem ID. |
| `SystemAnswer` | The system's answer. |

`Models & Types` · `AiDotNet.Audio.MusicAnalysis`

Represents a chord segment in audio.

## For Beginners

ChordSegment provides AI safety functionality. Default values follow the original paper settings.

## Properties

| Property | Summary |
|:-----|:--------|
| `Confidence` | Gets or sets the confidence score (0-1). |
| `Duration` | Gets the duration in seconds. |
| `EndTime` | Gets or sets the end time in seconds. |
| `Name` | Gets or sets the chord name (e.g., "C", "Am", "G7"). |
| `StartTime` | Gets or sets the start time in seconds. |

`Options & Configuration` · `AiDotNet.Audio.Speaker`

Configuration options for speaker verification.

## For Beginners

These options configure the SpeakerVerifier model. Default values follow the original paper settings.

| Constructor | Summary |
|:-----|:--------|
| `SpeakerVerifierOptions` | Initializes a new instance with default values. |
| `SpeakerVerifierOptions(SpeakerVerifierOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `IdentificationThreshold` | Gets or sets the threshold for identification (0-1). |
| `VerificationThreshold` | Gets or sets the threshold for verification (0-1). |

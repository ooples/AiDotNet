`Models & Types` · `AiDotNet.Audio.Whisper`

Result of Whisper transcription.

## For Beginners

WhisperResult provides AI safety functionality. Default values follow the original paper settings.

## Properties

| Property | Summary |
|:-----|:--------|
| `DetectedLanguage` | Gets or sets the detected language code. |
| `LanguageProbability` | Gets or sets the language detection probability. |
| `ProcessingTimeMs` | Gets or sets the processing time in milliseconds. |
| `Segments` | Gets or sets the segment-level timestamps. |
| `Text` | Gets or sets the transcribed text. |
| `Words` | Gets or sets the word-level timestamps if requested. |

## For Beginners

GenreClassificationResult provides AI safety functionality. Default values follow the original paper settings.

## Properties

| Property | Summary |
|:-----|:--------|
| `AllProbabilities` | Probabilities for all genres. |
| `Confidence` | Confidence score for predicted genre (0-1). |
| `Features` | Extracted features used for classification. |
| `PredictedGenre` | Most likely genre. |
| `TopPredictions` | Top K predictions with probabilities. |

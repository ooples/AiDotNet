`Models & Types` · `AiDotNet.Audio.Speaker`

Represents a speaker turn in diarization output.

## For Beginners

SpeakerTurn provides AI safety functionality. Default values follow the original paper settings.

## Properties

| Property | Summary |
|:-----|:--------|
| `Duration` | Gets the duration in seconds. |
| `EndTime` | Gets or sets the end time in seconds. |
| `SpeakerId` | Gets or sets the speaker ID label. |
| `SpeakerIndex` | Gets or sets the speaker index (0-based). |
| `StartTime` | Gets or sets the start time in seconds. |

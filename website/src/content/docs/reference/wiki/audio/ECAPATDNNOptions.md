`Options & Configuration` · `AiDotNet.Audio.LanguageIdentification`

Configuration options specific to ECAPA-TDNN language identification.

## Properties

| Property | Summary |
|:-----|:--------|
| `AttentionHeads` | Gets or sets the number of attention heads. |
| `Dilations` | Gets or sets the dilation factors for TDNN layers. |
| `TdnnChannels` | Gets or sets the number of TDNN channels. |
| `UseChannelAttention` | Gets or sets whether to use channel attention. |

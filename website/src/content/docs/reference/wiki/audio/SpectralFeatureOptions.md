`Options & Configuration` · `AiDotNet.Audio.Features`

Options for spectral feature extraction.

## For Beginners

These options configure the SpectralFeature model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `SpectralFeatureOptions` | Initializes a new instance with default values. |
| `SpectralFeatureOptions(SpectralFeatureOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `FeatureTypes` | Gets or sets which spectral features to extract. |
| `RolloffPercentage` | Gets or sets the rolloff percentage (0-1). |

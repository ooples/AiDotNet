## For Beginners

VerificationResult provides AI safety functionality. Default values follow the original paper settings.

|:-----|:--------|

| `ErrorMessage` | Gets or sets any error message. |
| `IsVerified` | Gets or sets whether the verification succeeded. |
| `Score` | Gets or sets the similarity score. |

`Models & Types` · `AiDotNet.ActiveLearning.Interfaces`

Represents a learning curve showing performance vs. number of labeled samples.

## Properties

| Property | Summary |
|:-----|:--------|
| `Accuracies` | Gets or sets the accuracy at each point. |
| `AreaUnderCurve` | Gets the area under the learning curve (AULC). |
| `Losses` | Gets or sets the loss at each point. |
| `SampleCounts` | Gets or sets the number of labeled samples at each point. |
| `ValidationAccuracies` | Gets or sets the validation accuracy at each point (if available). |
| `ValidationLosses` | Gets or sets the validation loss at each point (if available). |

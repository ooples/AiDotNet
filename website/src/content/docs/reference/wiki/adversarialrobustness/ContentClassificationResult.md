`Models & Types` · `AiDotNet.AdversarialRobustness.Safety`

Result of content classification by an ML model.

## Properties

| Property | Summary |
|:-----|:--------|
| `CategoryScores` | Gets or sets the scores for each category. |
| `DetectedCategories` | Gets or sets the list of categories that exceed the detection threshold. |
| `Explanation` | Gets or sets any explanation or reasoning for the classification. |
| `IsHarmful` | Gets or sets whether any harmful content was detected. |
| `OverallConfidence` | Gets or sets the overall confidence score for the classification. |
| `PrimaryCategory` | Gets or sets the primary detected category (the one with highest score). |
| `RecommendedAction` | Gets or sets the recommended action based on classification. |

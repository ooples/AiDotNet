`Models & Types` · `AiDotNet.Interpretability.Explainers`

Represents a matched prototype.

## Properties

| Property | Summary |
|:-----|:--------|
| `Distance` | Gets or sets the distance from the input to this prototype. |
| `IsSameClass` | Gets or sets whether this prototype has the same class as the input. |
| `Label` | Gets or sets the label of the prototype. |
| `PrototypeFeatures` | Gets or sets the feature values of the prototype. |
| `PrototypeIndex` | Gets or sets the index of the prototype in the prototype set. |
| `PrototypeName` | Gets or sets the name/ID of the prototype. |
| `Similarity` | Gets or sets the similarity (1 / (1 + distance)). |

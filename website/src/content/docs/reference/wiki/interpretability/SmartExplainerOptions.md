`Options & Configuration` · `AiDotNet.Interpretability`

Options for smart explainer selector.

## Properties

| Property | Summary |
|:-----|:--------|
| `EnableCaching` | Whether to enable caching of explanations. |
| `IntegrationSteps` | Number of integration steps for Integrated Gradients. |
| `MaxCacheSize` | Maximum cache size. |
| `MaxParallelism` | Maximum degree of parallelism. |
| `PreferDeepSHAP` | Prefer DeepSHAP over Integrated Gradients for neural networks. |
| `PreferLIME` | Prefer LIME over KernelSHAP for black-box models. |
| `UseParallelProcessing` | Whether to use parallel processing for batch explanations. |

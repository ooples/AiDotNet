`Models & Types` · `AiDotNet.AutoML.SearchSpace`

Defines the ResNet-based search space for neural architecture search.
Includes residual blocks, bottleneck blocks, and various skip connections.

## Properties

| Property | Summary |
|:-----|:--------|
| `BlockDepths` | Possible depths for residual blocks |
| `BottleneckRatio` | Bottleneck ratio for channel reduction |
| `GroupCount` | Number of groups for grouped convolutions (ResNeXt) |

`Models & Types` · `AiDotNet.AutoML`

Data transfer object for architecture JSON serialization.

## How It Works

This class provides a JSON-friendly structure for serializing architectures.
Tuples and complex types are converted to simple objects for better interoperability.

## Properties

| Property | Summary |
|:-----|:--------|
| `NodeChannels` | Gets or sets the node channel mappings. |
| `NodeCount` | Gets or sets the number of nodes in the architecture. |
| `Operations` | Gets or sets the list of operations in the architecture. |

`Models & Types` · `AiDotNet.AutoML.NAS`

Platform characteristics for hardware cost estimation.
Contains performance metrics like GFLOPS, memory bandwidth, and energy efficiency.

## Properties

| Property | Summary |
|:-----|:--------|
| `EnergyPerGBMemory` | Memory access energy cost in millijoules per GB. |
| `EnergyPerGFlop` | Energy consumption per GFLOP in millijoules. |
| `GFlops` | Peak computational throughput in GFLOPS (billions of floating-point operations per second). |
| `MemoryBandwidthGBps` | Memory bandwidth in GB/s. |

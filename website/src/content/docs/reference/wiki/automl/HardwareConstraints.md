Defines maximum latency, energy, and memory limits for architecture search.

| Property | Summary |
|:-----|:--------|
| `MaxEnergy` | Gets or sets the maximum energy constraint in millijoules. |
| `MaxLatency` | Gets or sets the maximum latency constraint in milliseconds. |
| `MaxMemory` | Gets or sets the maximum memory constraint in megabytes. |

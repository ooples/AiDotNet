`Models & Types` · `AiDotNet.AutoML`

Represents a compression configuration to be evaluated.

## Properties

| Property | Summary |
|:-----|:--------|
| `ErrorMessage` | Gets or sets any error message from the trial. |
| `FitnessScore` | Gets or sets the fitness score for this trial. |
| `Hyperparameters` | Gets or sets the hyperparameters for the compression technique. |
| `Metrics` | Gets or sets the resulting compression metrics. |
| `Success` | Gets or sets whether this trial completed successfully. |
| `Technique` | Gets or sets the compression technique used. |

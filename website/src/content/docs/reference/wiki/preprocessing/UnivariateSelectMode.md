Specifies the mode for GenericUnivariateSelect.

## Fields

| Field | Summary |
|:-----|:--------|
| `Fdr` | Select features based on false discovery rate (Benjamini-Hochberg). |
| `Fpr` | Select features based on false positive rate threshold. |
| `Fwe` | Select features based on family-wise error rate (Bonferroni). |
| `KBest` | Select the k highest-scoring features. |
| `Percentile` | Select the top percentile of features. |

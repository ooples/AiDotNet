`Options & Configuration` · `AiDotNet.Preprocessing.Document`

Options for document preprocessing.

## Properties

| Property | Summary |
|:-----|:--------|
| `ApplyBinarization` | Whether to apply binarization. |
| `ApplyDeskew` | Whether to apply deskewing. |
| `ApplyLayoutNormalization` | Whether to apply layout normalization. |
| `ApplyNoiseRemoval` | Whether to apply noise removal. |
| `BinarizationMethod` | Binarization method. |
| `DeskewMaxAngle` | Maximum deskew angle in degrees. |
| `NoiseRemovalMethod` | Noise removal method. |
| `NormalizeIntensity` | Whether to normalize intensity to 0-1 range. |
| `TargetHeight` | Target height for layout normalization. |
| `TargetWidth` | Target width for layout normalization. |

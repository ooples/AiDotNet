`Options & Configuration` · `AiDotNet.ContinualLearning.Strategies`

Configuration options for Progressive Neural Networks strategy.

## Properties

| Property | Summary |
|:-----|:--------|
| `AdapterDimension` | Gets or sets the adapter bottleneck dimension. |
| `HiddenUnitsPerColumn` | Gets or sets the number of hidden units per column. |
| `LateralInitScale` | Gets or sets the initialization scale for lateral connection weights. |
| `LateralScaling` | Gets or sets the lateral connection scaling factor. |
| `ShareOutputLayer` | Gets or sets whether to share the output layer across all columns. |
| `UseAdapters` | Gets or sets whether to use adapter layers for lateral connections. |

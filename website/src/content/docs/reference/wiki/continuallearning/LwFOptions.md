`Options & Configuration` · `AiDotNet.ContinualLearning.Strategies`

Configuration options for Learning without Forgetting.

## Properties

| Property | Summary |
|:-----|:--------|
| `DistillationWeight` | Gets or sets the weight for distillation loss relative to task loss. |
| `LossType` | Gets or sets the distillation loss type. |
| `Temperature` | Gets or sets the distillation temperature. |
| `UseFeatureDistillation` | Gets or sets whether to distill from intermediate layers. |
| `UseWarmup` | Gets or sets whether to use warmup for distillation weight. |
| `WarmupEpochs` | Gets or sets the number of warmup epochs. |

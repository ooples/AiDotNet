`Options & Configuration` · `AiDotNet.ComputerVision.Tracking`

Options for object tracking.

## Properties

| Property | Summary |
|:-----|:--------|
| `AppearanceWeight` | Weight for appearance similarity in cost matrix. |
| `ConfidenceThreshold` | Confidence threshold for detections. |
| `IouThreshold` | IoU threshold for detection-to-track association. |
| `MaxAge` | Maximum age (frames without detection) before track deletion. |
| `MaxCosineDistance` | Maximum cosine distance for appearance matching. |
| `MinHits` | Minimum hits before track is confirmed. |
| `UseAppearance` | Whether to use appearance features for matching. |

`Options & Configuration` · `AiDotNet.ComputerVision.Visualization`

Options for visualization.

## Properties

| Property | Summary |
|:-----|:--------|
| `BoxThickness` | Thickness of bounding boxes in pixels. |
| `ClassColors` | Custom class colors (classId -> RGB). |
| `FontScale` | Font scale for text. |
| `MaskOpacity` | Opacity for mask overlays (0-1). |
| `ShowConfidence` | Whether to show confidence scores. |
| `ShowLabels` | Whether to show class labels. |
| `ShowPolygons` | Whether to show polygon contours for text regions. |

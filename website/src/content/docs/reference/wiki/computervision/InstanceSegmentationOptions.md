`Options & Configuration` · `AiDotNet.ComputerVision.Segmentation.InstanceSegmentation`

Options for instance segmentation models.

## Properties

| Property | Summary |
|:-----|:--------|
| `Architecture` | Architecture to use for instance segmentation. |
| `ConfidenceThreshold` | Confidence threshold for detection. |
| `InputSize` | Input image size [height, width]. |
| `MaskResolution` | Mask resolution (output mask size). |
| `MaskThreshold` | Mask threshold for binarization. |
| `MaxDetections` | Maximum number of detections per image. |
| `NmsThreshold` | NMS threshold for suppression. |
| `NumClasses` | Number of classes to detect. |
| `UsePretrained` | Whether to use pretrained weights. |
| `WeightsUrl` | URL for pretrained weights. |

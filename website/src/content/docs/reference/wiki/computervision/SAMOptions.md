`Options & Configuration` · `AiDotNet.ComputerVision.Segmentation.Foundation`

Configuration options for the Segment Anything Model (SAM).

## For Beginners

SAM is Meta AI's foundation model for image segmentation.
Options inherit from NeuralNetworkOptions and can be extended with SAM-specific settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `SAMOptions` | Initializes a new instance with default values. |
| `SAMOptions(SAMOptions)` | Initializes a new instance by copying from another instance. |

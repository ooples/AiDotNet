`Options & Configuration` · `AiDotNet.ComputerVision.Detection.TextDetection`

Options for text detection models.

## Properties

| Property | Summary |
|:-----|:--------|
| `Architecture` | Text detection architecture to use. |
| `Backbone` | Backbone network type. |
| `BinaryThreshold` | Threshold for text/background binarization. |
| `ConfidenceThreshold` | Minimum confidence threshold for text detection. |
| `InputSize` | Input image size (height, width). |
| `MaxDetections` | Maximum number of detections. |
| `PolygonSimplificationEpsilon` | Polygon simplification threshold. |
| `Size` | Model size variant. |
| `UseMultiScale` | Whether to detect at multiple scales. |
| `UsePretrained` | Whether to use pretrained weights. |
| `WeightsUrl` | URL for pretrained weights. |

`Options & Configuration` · `AiDotNet.ComputerVision.Segmentation.InstanceSegmentation`

Configuration options for YOLOv12-Seg instance segmentation.

## For Beginners

These options configure the YOLOv12Seg model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `YOLOv12SegOptions` | Initializes a new instance with default values. |
| `YOLOv12SegOptions(YOLOv12SegOptions)` | Initializes a new instance by copying from another instance. |

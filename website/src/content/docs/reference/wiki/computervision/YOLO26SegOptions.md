`Options & Configuration` · `AiDotNet.ComputerVision.Segmentation.InstanceSegmentation`

Configuration options for YOLO26-Seg instance segmentation.

## For Beginners

These options configure the YOLO26Seg model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `YOLO26SegOptions` | Initializes a new instance with default values. |
| `YOLO26SegOptions(YOLO26SegOptions)` | Initializes a new instance by copying from another instance. |

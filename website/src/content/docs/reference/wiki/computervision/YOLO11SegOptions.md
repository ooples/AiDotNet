`Options & Configuration` · `AiDotNet.ComputerVision.Segmentation.InstanceSegmentation`

Configuration options for YOLO11-Seg instance segmentation.

## For Beginners

These options configure the YOLO11Seg model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `YOLO11SegOptions` | Initializes a new instance with default values. |
| `YOLO11SegOptions(YOLO11SegOptions)` | Initializes a new instance by copying from another instance. |

`Helpers & Utilities` · `AiDotNet.ComputerVision.Detection.Anchors`

Pre-defined anchor configurations for common models.

## Fields

| Field | Summary |
|:-----|:--------|
| `FPNStrides` | Standard strides for 5-level FPN (Faster R-CNN, RetinaNet). |
| `YOLOStrides` | Standard strides for YOLO models. |
| `YOLOv5Anchors` | YOLOv5 default anchors. |
| `YOLOv8Anchors` | YOLOv8 default anchors per level (not used in anchor-free YOLOv8, but available for compatibility). |

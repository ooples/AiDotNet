Configuration options for PIDNet real-time segmentation.

## For Beginners

These options configure the PIDNet model. Default values follow the original paper settings.

| Constructor | Summary |
|:-----|:--------|
| `PIDNetOptions` | Initializes a new instance with default values. |
| `PIDNetOptions(PIDNetOptions)` | Initializes a new instance by copying from another instance. |

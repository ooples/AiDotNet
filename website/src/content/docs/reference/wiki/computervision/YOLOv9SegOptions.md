`Options & Configuration` · `AiDotNet.ComputerVision.Segmentation.InstanceSegmentation`

Configuration options for YOLOv9-Seg instance segmentation.

## For Beginners

These options configure the YOLOv9Seg model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `YOLOv9SegOptions` | Initializes a new instance with default values. |
| `YOLOv9SegOptions(YOLOv9SegOptions)` | Initializes a new instance by copying from another instance. |

`Options & Configuration` · `AiDotNet.ComputerVision.Segmentation.Common`

Configuration for a segmentation dataset.

## Properties

| Property | Summary |
|:-----|:--------|
| `DefaultInputSize` | Default input size (height, width). |
| `IgnoreIndex` | Label index to ignore during evaluation (typically 255). |
| `Is3D` | Whether this dataset is 3D volumetric. |
| `Name` | Dataset name. |
| `NumClasses` | Number of classes. |
| `NumStuffClasses` | Number of "stuff" classes (for panoptic datasets). |
| `NumThingClasses` | Number of "thing" classes (for panoptic datasets). |
| `TaskTypes` | Supported task types. |

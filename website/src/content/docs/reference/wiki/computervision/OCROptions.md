`Options & Configuration` · `AiDotNet.ComputerVision.OCR`

Options for OCR models.

## Properties

| Property | Summary |
|:-----|:--------|
| `CharacterSet` | Character set/vocabulary for recognition. |
| `ConfidenceThreshold` | Minimum confidence threshold. |
| `CorrectSkew` | Whether to correct skewed text. |
| `DetectOrientation` | Whether to detect text orientation. |
| `DetectionModel` | Text detection model to use. |
| `GroupTextLines` | Whether to group text into lines/paragraphs. |
| `MaxRecognitionWidth` | Maximum input width for the recognition model. |
| `MaxSequenceLength` | Maximum sequence length for recognition. |
| `Mode` | OCR mode (scene text, document, or both). |
| `RecognitionHeight` | Input image height for the recognition model. |
| `RecognitionModel` | Text recognition model to use. |
| `SupportedLanguages` | Supported languages. |
| `UsePretrained` | Whether to use pretrained weights. |

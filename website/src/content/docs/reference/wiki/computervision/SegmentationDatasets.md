`Helpers & Utilities` · `AiDotNet.ComputerVision.Segmentation.Common`

Configuration for standard segmentation datasets.

## Properties

| Property | Summary |
|:-----|:--------|
| `ADE20K` | ADE20K dataset configuration (150 semantic classes, 20K train / 2K val images). |
| `BTCV` | BTCV multi-organ CT segmentation dataset (13 organs). |
| `COCOPanoptic` | COCO Panoptic dataset configuration (133 classes: 80 things + 53 stuff). |
| `Cityscapes` | Cityscapes dataset configuration (19 semantic classes, 8 instance classes). |
| `DAVIS2017` | DAVIS 2017 video object segmentation dataset. |
| `SA1B` | SA-1B Segment Anything dataset (11M images, 1B+ masks). |

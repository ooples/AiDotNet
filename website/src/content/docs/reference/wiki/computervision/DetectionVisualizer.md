`Models & Types` · `AiDotNet.ComputerVision.Visualization`

Visualizes object detection results on images.

## For Beginners

This class draws bounding boxes, class labels, and
confidence scores on images to visualize detection results.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `DetectionVisualizer(VisualizationOptions)` | Creates a new detection visualizer. |

## Methods

| Method | Summary |
|:-----|:--------|
| `DrawDetection(Tensor<>,Detection<>,String[])` | Draws a single detection on an image. |
| `Visualize(Tensor<>,DetectionResult<>,String[])` | Draws detection results on an image. |

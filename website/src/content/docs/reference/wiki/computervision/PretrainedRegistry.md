`Helpers & Utilities` · `AiDotNet.ComputerVision.Weights`

Registry of pre-trained model weights with their download URLs.

## Methods

| Method | Summary |
|:-----|:--------|
| `GetAvailableModels` | Gets all available pretrained model keys. |
| `GetDetectionModelKey(DetectionArchitecture,ModelSize,String)` | Generates the standard model key for detection models. |
| `GetUrl(String)` | Gets the URL for a pretrained model. |
| `Register(String,String)` | Registers a custom weight URL. |

`Options & Configuration` · `AiDotNet.ComputerVision.Segmentation.InstanceSegmentation`

Configuration options for the YOLOv8-Seg instance segmentation model.

## For Beginners

YOLOv8-Seg is a real-time instance segmentation model from Ultralytics.
Options inherit from NeuralNetworkOptions and can be extended with YOLO-specific settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `YOLOv8SegOptions` | Initializes a new instance with default values. |
| `YOLOv8SegOptions(YOLOv8SegOptions)` | Initializes a new instance by copying from another instance. |

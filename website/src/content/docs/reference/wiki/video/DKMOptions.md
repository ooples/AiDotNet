`Options & Configuration` · `AiDotNet.Video.Options`

Configuration options for the DKM (Dense Kernelized Matching) model.

## For Beginners

These options configure the DKM (Dense Kernelized Matching) model for dense correspondence estimation using learned kernels.
Default values follow the original paper recommendations.

`Options & Configuration` · `AiDotNet.Safety.Guardrails`

Configuration for guardrail modules.

## For Beginners

Use this to configure input/output guardrails including
length limits, topic restrictions, and custom rules.

## Properties

| Property | Summary |
|:-----|:--------|
| `DefaultAction` | Default action for guardrail violations. |
| `MaxInputLength` | Maximum allowed input length in characters. |
| `MaxOutputLength` | Maximum allowed output length in characters. |
| `TopicRestrictions` | Restricted topics to block. |

## Fields

| Field | Summary |
|:-----|:--------|
| `DefaultMaxInputLength` | Default maximum input length in characters. |
| `DefaultMaxOutputLength` | Default maximum output length in characters. |

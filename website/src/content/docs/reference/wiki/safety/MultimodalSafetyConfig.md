`Options & Configuration` · `AiDotNet.Safety.Multimodal`

Configuration for multimodal safety modules.

## For Beginners

Use this to configure cross-modal safety checking,
including text-image alignment and cross-modal attack detection.

## Properties

| Property | Summary |
|:-----|:--------|
| `CrossModalAttackDetection` | Whether to check for cross-modal attacks. |
| `MismatchThreshold` | Mismatch threshold for flagging cross-modal inconsistencies (0.0-1.0). |
| `TextImageAlignment` | Whether to check text-image alignment. |

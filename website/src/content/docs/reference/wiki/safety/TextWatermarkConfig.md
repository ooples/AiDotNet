`Options & Configuration` · `AiDotNet.Safety.Watermarking`

Configuration for text watermarking modules.

## For Beginners

Use this to configure text watermarking settings including
the watermark strength and detection threshold.

## Properties

| Property | Summary |
|:-----|:--------|
| `DetectionThreshold` | Detection threshold for watermark presence (0.0-1.0). |
| `Strength` | Watermark embedding strength (0.0-1.0). |
| `Technique` | Watermarking technique to use. |

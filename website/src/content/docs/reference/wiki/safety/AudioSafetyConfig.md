`Options & Configuration` · `AiDotNet.Safety.Audio`

Configuration for audio safety detection modules.

## For Beginners

Use this to configure audio safety detection including
deepfake detection, toxic speech detection, and voice protection settings.

## Properties

| Property | Summary |
|:-----|:--------|
| `DeepfakeThreshold` | Deepfake detection threshold (0.0-1.0). |
| `SampleRate` | Default sample rate for audio processing. |
| `ToxicityThreshold` | Toxicity detection threshold (0.0-1.0). |
| `VoiceProtection` | Whether to enable voice protection (anti-cloning). |

## Properties

| Property | Summary |
|:-----|:--------|
| `Description` | Gets or sets a description of the test case for reporting. |
| `ExpectedCategory` | Gets or sets the expected safety category (for per-category metrics). |
| `ExpectedUnsafe` | Gets or sets whether this test case is expected to be flagged as unsafe. |
| `Text` | Gets or sets the test text to evaluate. |

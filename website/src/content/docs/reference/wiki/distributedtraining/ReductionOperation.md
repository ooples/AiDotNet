`Enums` · `AiDotNet.DistributedTraining`

Defines the supported reduction operations for collective communication.

## For Beginners

These are different ways to combine values from multiple processes.

## Fields

| Field | Summary |
|:-----|:--------|
| `Average` | Compute average (sum divided by count) |
| `Max` | Take the maximum value |
| `Min` | Take the minimum value |
| `Product` | Multiply all values together |
| `Sum` | Add all values together |

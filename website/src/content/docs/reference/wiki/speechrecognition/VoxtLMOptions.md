`Options & Configuration` · `AiDotNet.SpeechRecognition.Specialized`

Options for VoxtLM: decoder-only model for speech-text joint training.

## For Beginners

These options configure the VoxtLM model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

| Constructor | Summary |

`Options & Configuration` · `AiDotNet.SpeechRecognition.LLMIntegrated`

Options for Phi-4 Audio: Microsoft's efficient speech-language model.

## For Beginners

These options configure the Phi4Audio model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `Phi4AudioOptions` | Initializes a new instance with default values. |
| `Phi4AudioOptions(Phi4AudioOptions)` | Initializes a new instance by copying from another instance. |

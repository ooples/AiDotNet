`Options & Configuration` · `AiDotNet.SpeechRecognition.ConformerFamily`

Options for Conformer-Transducer variant (Conformer encoder + RNN-T/TDT decoder).

## For Beginners

These options configure the ConformerTransducer model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

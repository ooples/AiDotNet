`Options & Configuration` · `AiDotNet.SpeechRecognition.NeMo`

Options for Canary-Qwen (NVIDIA NeMo, 2025): multilingual ASR + translation with Qwen-2.5 LLM decoder.

## For Beginners

These options configure the CanaryQwen model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `CanaryQwenOptions` | Initializes a new instance with default values. |
| `CanaryQwenOptions(CanaryQwenOptions)` | Initializes a new instance by copying from another instance. |

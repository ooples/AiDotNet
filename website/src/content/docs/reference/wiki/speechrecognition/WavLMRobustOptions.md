`Options & Configuration` · `AiDotNet.SpeechRecognition.Robust`

Options for WavLM-Robust: noise-robust speech recognition via denoising pre-training.

## For Beginners

These options configure the WavLMRobust model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

`Options & Configuration` · `AiDotNet.SpeechRecognition.Robust`

Options for SpeechBrain: open-source speech processing toolkit ASR models.

## For Beginners

These options configure the SpeechBrain model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `SpeechBrainOptions` | Initializes a new instance with default values. |
| `SpeechBrainOptions(SpeechBrainOptions)` | Initializes a new instance by copying from another instance. |

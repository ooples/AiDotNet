`Options & Configuration` · `AiDotNet.SpeechRecognition.ProprietaryAPI`

Options for Azure Speech Service: Microsoft's enterprise ASR platform.

## For Beginners

These options configure the AzureSpeechSTT model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `AzureSpeechSTTOptions` | Initializes a new instance with default values. |
| `AzureSpeechSTTOptions(AzureSpeechSTTOptions)` | Initializes a new instance by copying from another instance. |

`Options & Configuration` · `AiDotNet.SpeechRecognition.Foundation`

Options for Wav2Vec 2.0 ASR: Meta's self-supervised speech representation model.

## For Beginners

These options configure the Wav2Vec2ASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `Wav2Vec2ASROptions` | Initializes a new instance with default values. |
| `Wav2Vec2ASROptions(Wav2Vec2ASROptions)` | Initializes a new instance by copying from another instance. |

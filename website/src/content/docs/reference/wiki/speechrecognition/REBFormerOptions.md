`Options & Configuration` · `AiDotNet.SpeechRecognition.ConformerFamily`

Options for REBFormer (RWKV-Enhanced E-Branchformer, Song et al., 2025).

## For Beginners

These options configure the REBFormer model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `REBFormerOptions` | Initializes a new instance with default values. |
| `REBFormerOptions(REBFormerOptions)` | Initializes a new instance by copying from another instance. |

`Options & Configuration` · `AiDotNet.SpeechRecognition.Foundation`

Options for Data2Vec ASR: general-purpose self-supervised learning.

## For Beginners

These options configure the Data2VecASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `Data2VecASROptions` | Initializes a new instance with default values. |
| `Data2VecASROptions(Data2VecASROptions)` | Initializes a new instance by copying from another instance. |

`Options & Configuration` · `AiDotNet.SpeechRecognition.Foundation`

Options for BEST-RQ: BERT-based self-supervised learning with random-projection quantizer.

## For Beginners

These options configure the BESTRQ model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `BESTRQOptions` | Initializes a new instance with default values. |
| `BESTRQOptions(BESTRQOptions)` | Initializes a new instance by copying from another instance. |

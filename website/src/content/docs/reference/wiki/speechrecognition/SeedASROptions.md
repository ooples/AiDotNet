`Options & Configuration` · `AiDotNet.SpeechRecognition.LLMIntegrated`

Options for Seed-ASR: ByteDance's large-scale multilingual ASR.

## For Beginners

These options configure the SeedASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `SeedASROptions` | Initializes a new instance with default values. |
| `SeedASROptions(SeedASROptions)` | Initializes a new instance by copying from another instance. |

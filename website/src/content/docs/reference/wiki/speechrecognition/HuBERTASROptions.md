`Options & Configuration` · `AiDotNet.SpeechRecognition.Foundation`

Options for HuBERT ASR: Hidden-Unit BERT for self-supervised speech representation.

## For Beginners

These options configure the HuBERTASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `HuBERTASROptions` | Initializes a new instance with default values. |
| `HuBERTASROptions(HuBERTASROptions)` | Initializes a new instance by copying from another instance. |

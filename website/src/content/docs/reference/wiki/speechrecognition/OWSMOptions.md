`Options & Configuration` · `AiDotNet.SpeechRecognition.Multilingual`

Options for OWSM: Open Whisper-Style Speech Model.

## For Beginners

These options configure the OWSM model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `OWSMOptions` | Initializes a new instance with default values. |
| `OWSMOptions(OWSMOptions)` | Initializes a new instance by copying from another instance. |

`Options & Configuration` · `AiDotNet.SpeechRecognition.ConformerFamily`

Options for CIF (Continuous Integrate-and-Fire) encoder for soft monotonic alignment (Dong & Xu, 2020).

## For Beginners

These options configure the CIFEncoder model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `CIFEncoderOptions` | Initializes a new instance with default values. |
| `CIFEncoderOptions(CIFEncoderOptions)` | Initializes a new instance by copying from another instance. |

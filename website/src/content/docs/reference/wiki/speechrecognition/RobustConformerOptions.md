`Options & Configuration` · `AiDotNet.SpeechRecognition.Robust`

Options for Robust Conformer: adversarially-trained Conformer for noisy ASR.

## For Beginners

These options configure the RobustConformer model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `RobustConformerOptions` | Initializes a new instance with default values. |
| `RobustConformerOptions(RobustConformerOptions)` | Initializes a new instance by copying from another instance. |

Options for Granite Speech: IBM's enterprise speech-language model.

## For Beginners

These options configure the GraniteSpeech model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

| Constructor | Summary |

| `GraniteSpeechOptions` | Initializes a new instance with default values. |

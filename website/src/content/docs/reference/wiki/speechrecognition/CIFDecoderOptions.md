`Options & Configuration` · `AiDotNet.SpeechRecognition.CTCVariants`

Options for CIF: Continuous Integrate-and-Fire mechanism for non-autoregressive ASR.

## For Beginners

These options configure the CIFDecoder model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `CIFDecoderOptions` | Initializes a new instance with default values. |
| `CIFDecoderOptions(CIFDecoderOptions)` | Initializes a new instance by copying from another instance. |

`Options & Configuration` · `AiDotNet.SpeechRecognition.LLMIntegrated`

Options for OLMo-ASR: open language model adapted for speech recognition.

## For Beginners

These options configure the OLMoASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `OLMoASROptions` | Initializes a new instance with default values. |
| `OLMoASROptions(OLMoASROptions)` | Initializes a new instance by copying from another instance. |

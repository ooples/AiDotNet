Options for Parakeet-RNNT (NVIDIA NeMo, 2024): 1.1B Conformer with RNN-Transducer decoder.

## For Beginners

These options configure the ParakeetRNNT model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

`Options & Configuration` · `AiDotNet.SpeechRecognition.Multilingual`

Options for Chirp 3: latest-generation multilingual ASR.

## For Beginners

These options configure the Chirp3 model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `Chirp3Options` | Initializes a new instance with default values. |

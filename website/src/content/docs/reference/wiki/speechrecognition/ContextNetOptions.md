`Options & Configuration` · `AiDotNet.SpeechRecognition.ConformerFamily`

Options for ContextNet CNN encoder with squeeze-and-excitation (Han et al., 2020).

## For Beginners

These options configure the ContextNet model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `ContextNetOptions` | Initializes a new instance with default values. |
| `ContextNetOptions(ContextNetOptions)` | Initializes a new instance by copying from another instance. |

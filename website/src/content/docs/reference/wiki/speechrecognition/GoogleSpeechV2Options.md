`Options & Configuration` · `AiDotNet.SpeechRecognition.ProprietaryAPI`

Options for Google Cloud Speech-to-Text V2: production cloud ASR API.

## For Beginners

These options configure the GoogleSpeechV2 model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

| Constructor | Summary |
|:-----|:--------|
| `GoogleSpeechV2Options` | Initializes a new instance with default values. |

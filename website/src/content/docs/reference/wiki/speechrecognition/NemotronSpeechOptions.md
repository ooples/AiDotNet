`Options & Configuration` · `AiDotNet.SpeechRecognition.NeMo`

Options for Nemotron-Speech (NVIDIA, 2025): multi-task ASR model with Nemotron LLM backbone.

## For Beginners

These options configure the NemotronSpeech model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `NemotronSpeechOptions` | Initializes a new instance with default values. |
| `NemotronSpeechOptions(NemotronSpeechOptions)` | Initializes a new instance by copying from another instance. |

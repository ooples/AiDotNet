`Options & Configuration` · `AiDotNet.SpeechRecognition.Multilingual`

Options for Omnilingual ASR: universal multilingual speech recognition.

## For Beginners

These options configure the OmnilangualASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `OmnilangualASROptions` | Initializes a new instance with default values. |
| `OmnilangualASROptions(OmnilangualASROptions)` | Initializes a new instance by copying from another instance. |

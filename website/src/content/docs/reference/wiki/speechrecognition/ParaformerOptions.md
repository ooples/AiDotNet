`Options & Configuration` · `AiDotNet.SpeechRecognition.AlibabaASR`

Options for Paraformer: fast and accurate parallel Transformer for non-autoregressive ASR.

## For Beginners

These options configure the Paraformer model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `ParaformerOptions` | Initializes a new instance with default values. |
| `ParaformerOptions(ParaformerOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `Vocabulary` | Optional vocabulary mapping token IDs to text tokens (e.g., SentencePiece vocabulary). |

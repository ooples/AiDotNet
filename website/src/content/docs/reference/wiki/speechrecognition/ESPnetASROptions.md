`Options & Configuration` · `AiDotNet.SpeechRecognition.Robust`

Options for ESPnet ASR: end-to-end speech processing toolkit models.

## For Beginners

These options configure the ESPnetASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `ESPnetASROptions` | Initializes a new instance with default values. |
| `ESPnetASROptions(ESPnetASROptions)` | Initializes a new instance by copying from another instance. |

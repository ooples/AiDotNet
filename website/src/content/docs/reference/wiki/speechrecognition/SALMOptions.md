`Options & Configuration` · `AiDotNet.SpeechRecognition.LLMIntegrated`

Options for SALM: speech-augmented language model for ASR.

## For Beginners

These options configure the SALM model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `SALMOptions` | Initializes a new instance with default values. |
| `SALMOptions(SALMOptions)` | Initializes a new instance by copying from another instance. |

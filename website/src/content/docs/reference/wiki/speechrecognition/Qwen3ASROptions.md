`Options & Configuration` · `AiDotNet.SpeechRecognition.AlibabaASR`

Options for Qwen3-ASR: LLM-integrated ASR with Qwen3 decoder.

## For Beginners

These options configure the Qwen3ASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `Qwen3ASROptions` | Initializes a new instance with default values. |
| `Qwen3ASROptions(Qwen3ASROptions)` | Initializes a new instance by copying from another instance. |

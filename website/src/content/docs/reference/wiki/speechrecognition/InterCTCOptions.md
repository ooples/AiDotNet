`Options & Configuration` · `AiDotNet.SpeechRecognition.CTCVariants`

Options for Inter-CTC: intermediate CTC loss for deep encoder training.

## For Beginners

These options configure the InterCTC model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `InterCTCOptions` | Initializes a new instance with default values. |
| `InterCTCOptions(InterCTCOptions)` | Initializes a new instance by copying from another instance. |

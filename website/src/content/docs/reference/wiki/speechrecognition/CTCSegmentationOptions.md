Options for CTC Segmentation: forced alignment using CTC posteriors.

## For Beginners

These options configure the CTCSegmentation model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

| Constructor | Summary |
|:-----|:--------|
| `CTCSegmentationOptions` | Initializes a new instance with default values. |

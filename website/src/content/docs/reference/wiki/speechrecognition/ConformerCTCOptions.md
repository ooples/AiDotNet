`Options & Configuration` · `AiDotNet.SpeechRecognition.ConformerFamily`

Options for Conformer-CTC variant (Conformer encoder + CTC head, no decoder).

## For Beginners

These options configure the ConformerCTC model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `ConformerCTCOptions` | Initializes a new instance with default values. |
| `ConformerCTCOptions(ConformerCTCOptions)` | Initializes a new instance by copying from another instance. |

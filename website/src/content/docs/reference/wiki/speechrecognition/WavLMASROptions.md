`Options & Configuration` · `AiDotNet.SpeechRecognition.Foundation`

Options for WavLM ASR: speech pre-training with denoising and attention.

## For Beginners

These options configure the WavLMASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `WavLMASROptions` | Initializes a new instance with default values. |
| `WavLMASROptions(WavLMASROptions)` | Initializes a new instance by copying from another instance. |

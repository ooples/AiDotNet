`Options & Configuration` · `AiDotNet.SpeechRecognition.LLMIntegrated`

Options for SpeechGPT: speech-language model for multi-modal conversation.

## For Beginners

These options configure the SpeechGPTASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `SpeechGPTASROptions` | Initializes a new instance with default values. |
| `SpeechGPTASROptions(SpeechGPTASROptions)` | Initializes a new instance by copying from another instance. |

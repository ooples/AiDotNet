`Options & Configuration` · `AiDotNet.SpeechRecognition.Multilingual`

Options for XLS-R: cross-lingual speech representation at scale.

## For Beginners

These options configure the XLSR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `XLSROptions` | Initializes a new instance with default values. |
| `XLSROptions(XLSROptions)` | Initializes a new instance by copying from another instance. |

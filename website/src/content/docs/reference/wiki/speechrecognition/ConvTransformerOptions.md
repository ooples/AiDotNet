`Options & Configuration` · `AiDotNet.SpeechRecognition.ConformerFamily`

Options for ConvTransformer (convolution-augmented Transformer for ASR, 2019).

## For Beginners

These options configure the ConvTransformer model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `ConvTransformerOptions` | Initializes a new instance with default values. |
| `ConvTransformerOptions(ConvTransformerOptions)` | Initializes a new instance by copying from another instance. |

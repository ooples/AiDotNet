`Options & Configuration` · `AiDotNet.SpeechRecognition.Streaming`

Options for Emformer-RNNT: efficient memory Transformer for streaming ASR.

## For Beginners

These options configure the EmformerRNNT model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `EmformerRNNTOptions` | Initializes a new instance with default values. |

`Options & Configuration` · `AiDotNet.SpeechRecognition.Multilingual`

Options for Chirp: Google's production multilingual ASR.

## For Beginners

These options configure the Chirp model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `ChirpOptions` | Initializes a new instance with default values. |
| `ChirpOptions(ChirpOptions)` | Initializes a new instance by copying from another instance. |

`Options & Configuration` · `AiDotNet.SpeechRecognition.Robust`

Options for AV-HuBERT: audio-visual speech recognition with HuBERT.

## For Beginners

These options configure the AVHuBERT model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `AVHuBERTOptions` | Initializes a new instance with default values. |
| `AVHuBERTOptions(AVHuBERTOptions)` | Initializes a new instance by copying from another instance. |

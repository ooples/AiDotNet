`Options & Configuration` · `AiDotNet.SpeechRecognition.ProprietaryAPI`

Options for AWS Transcribe: Amazon's scalable speech recognition service.

## For Beginners

These options configure the AWSTranscribe model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `AWSTranscribeOptions` | Initializes a new instance with default values. |
| `AWSTranscribeOptions(AWSTranscribeOptions)` | Initializes a new instance by copying from another instance. |

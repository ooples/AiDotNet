Options for UniSpeech: unified pre-training for speech representation learning.

## For Beginners

These options configure the UniSpeech model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

| Constructor | Summary |

| `UniSpeechOptions` | Initializes a new instance with default values. |

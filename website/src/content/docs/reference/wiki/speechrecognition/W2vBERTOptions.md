`Options & Configuration` · `AiDotNet.SpeechRecognition.Foundation`

Options for W2v-BERT: combining contrastive learning and masked language modeling for speech.

## For Beginners

These options configure the W2vBERT model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `W2vBERTOptions` | Initializes a new instance with default values. |
| `W2vBERTOptions(W2vBERTOptions)` | Initializes a new instance by copying from another instance. |

`Options & Configuration` · `AiDotNet.SpeechRecognition.LLMIntegrated`

Options for Samba-ASR: Mamba-based state-space model for speech recognition.

## For Beginners

These options configure the SambaASR model. Default values follow the original paper's recommended settings for optimal speech recognition accuracy.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `SambaASROptions` | Initializes a new instance with default values. |
| `SambaASROptions(SambaASROptions)` | Initializes a new instance by copying from another instance. |

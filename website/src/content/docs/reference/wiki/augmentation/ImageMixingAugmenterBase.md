`Base Classes` · `AiDotNet.Augmentation.Image`

Base class for image augmentations that mix multiple images together.

## How It Works

Use this base class for augmentations like MixUp and CutMix that
combine two or more images and their labels.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `ImageMixingAugmenterBase(Double,Double)` | Initializes a new image mixing augmentation. |

`Models & Types` · `AiDotNet.Augmentation.Image`

Simulates JPEG compression artifacts by quantizing DCT coefficients.

## How It Works

Simulates the blocking and ringing artifacts from low-quality JPEG compression
using 8x8 DCT block quantization. Lower quality values produce stronger artifacts.

`Models & Types` · `AiDotNet.Augmentation`

Represents a recommendation for an augmentation with its configuration.

## Properties

| Property | Summary |
|:-----|:--------|
| `AugmentationType` | Gets or sets the augmentation type name. |
| `ConfidenceScore` | Gets or sets the confidence score of this recommendation (0.0 to 1.0). |
| `IncompatibleWith` | Gets or sets augmentations this is incompatible with. |
| `IsCritical` | Gets or sets whether this augmentation is critical for the task. |
| `Priority` | Gets or sets the priority order (lower = higher priority). |
| `Reason` | Gets or sets the reason for this recommendation. |
| `RecommendedParameters` | Gets or sets the recommended parameters. |
| `RecommendedProbability` | Gets or sets the recommended probability (0.0 to 1.0). |

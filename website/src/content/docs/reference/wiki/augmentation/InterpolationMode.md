Specifies the interpolation method for image resizing.

## Fields

| Field | Summary |
|:-----|:--------|
| `Area` | Area-based resampling (good for downscaling). |
| `Bicubic` | Bicubic interpolation (smoother, slower). |
| `Bilinear` | Bilinear interpolation (good balance). |
| `Lanczos` | Lanczos resampling (high quality, slowest). |
| `Nearest` | Nearest neighbor interpolation (fastest, blocky). |

`Helpers & Utilities` · `AiDotNet.Augmentation`

Utility for auto-detecting data modality from input types.

## For Beginners

This helper automatically determines what kind of data
you're working with (images, tables, audio, etc.) so the right augmentations can be applied.

## Methods

| Method | Summary |
|:-----|:--------|
| `Detect` | Detects the data modality from the input type. |

`Base Classes` · `AiDotNet.Augmentation.Image`

Base class for image augmentations that transform spatial targets.

## How It Works

Use this base class for augmentations that need to also transform
bounding boxes, keypoints, or segmentation masks along with the image.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `SpatialImageAugmenterBase(Double)` | Initializes a new spatial image augmentation. |

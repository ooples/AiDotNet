`Models & Types` · `AiDotNet.Augmentation`

Represents a hyperparameter definition for AutoML search.

## Properties

| Property | Summary |
|:-----|:--------|
| `Choices` | Gets or sets the categorical choices (for categorical type). |
| `Conditions` | Gets or sets conditional dependencies (e.g., only active if another param has specific value). |
| `DefaultValue` | Gets or sets the default value. |
| `Description` | Gets or sets the description of this parameter. |
| `IsRequired` | Gets or sets whether this parameter is required. |
| `MaxValue` | Gets or sets the maximum value (for continuous/integer/log-scale). |
| `MinValue` | Gets or sets the minimum value (for continuous/integer/log-scale). |
| `Name` | Gets or sets the parameter name. |
| `Type` | Gets or sets the parameter type. |

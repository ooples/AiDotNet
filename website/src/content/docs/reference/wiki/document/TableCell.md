`Models & Types` · `AiDotNet.Document`

Represents a single cell in a table.

## Properties

| Property | Summary |
|:-----|:--------|
| `BoundingBox` | Gets the bounding box of the cell. |
| `ColSpan` | Gets the column span (number of columns this cell covers). |
| `Column` | Gets the column index (0-based). |
| `Confidence` | Gets the confidence for this cell's detection. |
| `IsHeader` | Gets whether this is a header cell. |
| `Row` | Gets the row index (0-based). |
| `RowSpan` | Gets the row span (number of rows this cell covers). |
| `Text` | Gets the text content of the cell. |

`Models & Types` · `AiDotNet.Document`

Represents a block (paragraph or region) of recognized text.

## Properties

| Property | Summary |
|:-----|:--------|
| `AverageConfidence` | Gets the average confidence for text in this block. |
| `BlockType` | Gets the detected block type (text, table, figure, etc.). |
| `BoundingBox` | Gets the bounding box as [x1, y1, x2, y2]. |
| `Lines` | Gets the lines in this block. |
| `Text` | Gets the full text of the block. |

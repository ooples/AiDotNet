`Models & Types` · `AiDotNet.Document`

Represents a region in the document that provides evidence for an answer.

## Properties

| Property | Summary |
|:-----|:--------|
| `BoundingBox` | Gets the bounding box of the evidence region. |
| `PageIndex` | Gets the page number where the evidence was found (0-indexed). |
| `Relevance` | Gets how relevant this evidence is to the answer (0-1). |
| `Text` | Gets the text content of the evidence. |

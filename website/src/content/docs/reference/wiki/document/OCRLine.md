`Models & Types` · `AiDotNet.Document`

Represents a line of recognized text.

## Properties

| Property | Summary |
|:-----|:--------|
| `AverageConfidence` | Gets the average confidence for words in this line. |
| `BlockIndex` | Gets the block index this line belongs to. |
| `BoundingBox` | Gets the bounding box as [x1, y1, x2, y2]. |
| `Text` | Gets the full text of the line. |
| `Words` | Gets the words in this line. |

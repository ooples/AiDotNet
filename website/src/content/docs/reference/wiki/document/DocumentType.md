`Enums` · `AiDotNet.Document`

Types of documents supported by document AI models.

## For Beginners

Document AI models can be specialized for different document types.
Use these flags to specify which types a model supports or to filter processing.
Multiple types can be combined using the | operator.

## Fields

| Field | Summary |
|:-----|:--------|
| `All` | All document types supported. |
| `BusinessDocument` | Business documents like invoices, receipts, purchase orders, and contracts. |
| `Form` | Structured forms with fields, checkboxes, and signatures. |
| `Handwritten` | Handwritten documents and notes. |
| `Infographic` | Charts, diagrams, and infographics. |
| `Letter` | Letters and general correspondence. |
| `None` | No specific document type. |
| `Report` | Business and financial reports with complex layouts. |
| `ScannedDocument` | General scanned documents. |
| `ScientificPaper` | Scientific and academic papers with figures, tables, and citations. |
| `WebPage` | Screenshots of web pages. |

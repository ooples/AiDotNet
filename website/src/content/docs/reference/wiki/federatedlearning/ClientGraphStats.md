`Models & Types` · `AiDotNet.FederatedLearning.Graph`

Statistics about a client's local subgraph, used for graph-aware aggregation weighting.

## Properties

| Property | Summary |
|:-----|:--------|
| `AverageDegree` | Gets or sets the average node degree in the subgraph. |
| `BorderNodeCount` | Gets or sets the number of cross-client border nodes. |
| `EdgeCount` | Gets or sets the number of edges in the client's subgraph. |
| `LabeledNodeCount` | Gets or sets the number of labeled nodes (for supervised tasks). |
| `NodeCount` | Gets or sets the number of nodes in the client's subgraph. |

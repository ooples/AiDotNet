`Options & Configuration` · `AiDotNet.FederatedLearning.Alignment`

Configuration options for Federated DPO.

## Properties

| Property | Summary |
|:-----|:--------|
| `Beta` | Gets or sets the DPO temperature (beta). |
| `LearningRate` | Gets or sets the learning rate for DPO training. |
| `LoRARank` | Gets or sets the LoRA rank if UseLoRA is true. |
| `LocalEpochs` | Gets or sets the number of local DPO epochs per round. |
| `UseLoRA` | Gets or sets whether to use LoRA for DPO training. |

## Fields

| Field | Summary |
|:-----|:--------|
| `_beta` | Gets or sets the DPO temperature (beta). |

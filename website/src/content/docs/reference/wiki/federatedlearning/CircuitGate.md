`Models & Types` · `AiDotNet.FederatedLearning.MPC`

Represents a single gate in a boolean circuit.

## Properties

| Property | Summary |
|:-----|:--------|
| `InputWire0` | Gets or sets the index of the first input wire. |
| `InputWire1` | Gets or sets the index of the second input wire (-1 for unary gates like NOT). |
| `OutputWire` | Gets or sets the index of the output wire. |
| `Type` | Gets or sets the type of gate (AND, XOR, NOT, OR). |

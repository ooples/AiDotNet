`Models & Types` · `AiDotNet.FederatedLearning.Graph`

Represents a sampled subgraph from neighborhood sampling.

## Properties

| Property | Summary |
|:-----|:--------|
| `Adjacency` | Gets or sets the adjacency matrix of the sampled subgraph. |
| `NodeFeatures` | Gets or sets the node features of the sampled subgraph. |
| `NodeMapping` | Gets or sets the mapping from sampled node indices to original node indices. |
| `TargetNodeIndices` | Gets or sets the indices of the target nodes within the sampled subgraph. |

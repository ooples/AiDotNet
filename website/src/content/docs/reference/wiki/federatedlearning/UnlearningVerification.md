`Models & Types` · `AiDotNet.FederatedLearning.Vertical`

Contains the results of an unlearning verification check.

## Properties

| Property | Summary |
|:-----|:--------|
| `DivergenceScore` | Gets or sets the prediction divergence score (how much predictions changed after unlearning). |
| `EntitiesUnlearned` | Gets or sets the number of entities that were unlearned. |
| `Passed` | Gets or sets whether the unlearning verification passed. |
| `Threshold` | Gets or sets the threshold used for the pass/fail decision. |

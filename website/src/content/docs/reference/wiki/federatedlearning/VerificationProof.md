| Property | Summary |
|:-----|:--------|

| `Commitment` | Gets or sets the commitment to the proven value. |
| `ProofData` | Gets or sets the proof data. |
| `ProofSystem` | Gets or sets the proof system used. |
| `Round` | Gets or sets the round number when the proof was generated. |

Types of boolean gates supported in garbled circuits.

## Fields

| Field | Summary |
|:-----|:--------|
| `And` | AND gate — output is 1 only if both inputs are 1. |
| `Not` | NOT gate (unary) — output is the inverse of input. |
| `Or` | OR gate — output is 1 if either input is 1. |
| `Xor` | XOR gate — output is 1 if inputs differ. |

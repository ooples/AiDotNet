`Options & Configuration` · `AiDotNet.FederatedLearning.Alignment`

Configuration options for the OpenFedLLM pipeline.

## Properties

| Property | Summary |
|:-----|:--------|
| `AlignmentLR` | Gets or sets the learning rate for alignment. |
| `AlignmentRounds` | Gets or sets the number of alignment rounds. |
| `InstructionTuningLR` | Gets or sets the learning rate for instruction tuning. |
| `InstructionTuningRounds` | Gets or sets the number of instruction tuning rounds. |
| `LoRARank` | Gets or sets the LoRA rank for instruction tuning. |
| `UseDPO` | Gets or sets whether to use DPO (true) or RLHF (false) for alignment. |

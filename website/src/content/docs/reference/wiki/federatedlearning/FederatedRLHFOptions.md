`Options & Configuration` · `AiDotNet.FederatedLearning.Alignment`

Configuration options for Federated RLHF.

## Properties

| Property | Summary |
|:-----|:--------|
| `AdapterOnlyRewardModel` | Gets or sets whether to use adapter-only reward models. |
| `ClipRange` | Gets or sets the PPO clip range. |
| `KLCoefficient` | Gets or sets the KL divergence penalty coefficient (beta). |
| `PPOEpochs` | Gets or sets the number of PPO epochs per round. |
| `RewardModelLearningRate` | Gets or sets the reward model learning rate. |

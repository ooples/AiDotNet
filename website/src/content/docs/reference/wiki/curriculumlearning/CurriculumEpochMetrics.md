`Models & Types` · `AiDotNet.CurriculumLearning.Interfaces`

Metrics from a curriculum learning epoch.

## Properties

| Property | Summary |
|:-----|:--------|
| `Epoch` | Gets or sets the current epoch number. |
| `Improved` | Gets or sets whether this epoch showed improvement. |
| `LossImprovement` | Gets or sets the improvement in loss from previous epoch. |
| `Phase` | Gets or sets the current phase number. |
| `SampleLosses` | Gets or sets per-sample losses. |
| `SamplesUsed` | Gets or sets the number of samples used in this epoch. |
| `TrainingAccuracy` | Gets or sets the training accuracy (for classification). |
| `TrainingLoss` | Gets or sets the average training loss. |
| `ValidationAccuracy` | Gets or sets the validation accuracy (if applicable). |
| `ValidationLoss` | Gets or sets the validation loss (if validation data provided). |

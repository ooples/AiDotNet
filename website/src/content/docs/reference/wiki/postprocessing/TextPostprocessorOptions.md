`Options & Configuration` · `AiDotNet.Postprocessing.Document`

Options for text postprocessing.

## Properties

| Property | Summary |
|:-----|:--------|
| `ApplySpellCorrection` | Whether to apply spell correction. |
| `FixCommonOcrErrors` | Whether to fix common OCR errors. |
| `MergeBrokenLines` | Whether to merge incorrectly broken lines. |
| `NormalizeCharacters` | Whether to normalize special characters. |
| `NormalizeWhitespace` | Whether to normalize whitespace. |
| `RemoveControlCharacters` | Whether to remove control characters. |
| `RemoveDuplicateSpaces` | Whether to remove duplicate spaces. |

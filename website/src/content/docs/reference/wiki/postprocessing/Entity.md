`Models & Types` · `AiDotNet.Postprocessing.Document`

Represents an extracted entity from document text.

## Properties

| Property | Summary |
|:-----|:--------|
| `Attributes` | Additional attributes for the entity. |
| `CanonicalName` | Canonical name for the entity. |
| `Confidence` | Confidence score for the extraction. |
| `EndIndex` | End index in the original text. |
| `LinkedEntity` | Linked entity from knowledge base. |
| `NormalizedValue` | Normalized value (for dates, money, etc.). |
| `StartIndex` | Start index in the original text. |
| `Text` | The original text of the entity. |
| `Type` | The type of entity. |

`Options & Configuration` · `AiDotNet.Diffusion.ThreeD`

Configuration for MVDream model.

## Properties

| Property | Summary |
|:-----|:--------|
| `CameraDistance` | Camera distance from object center. |
| `Default` | Gets the default configuration. |
| `DefaultElevation` | Default elevation angle in degrees. |
| `DefaultNumViews` | Default number of views to generate. |
| `ImageSize` | Image resolution for each view. |
| `UseMultiViewAttention` | Whether to use multi-view attention. |

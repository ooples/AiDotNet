Oasis playable AI game via next-frame prediction.

## For Beginners

Oasis is an AI-powered Minecraft engine that generates the next video frame based on player actions. It creates a fully playable Minecraft-like experience through real-time next-frame prediction.

## How It Works

**References:**

- Reference: Oasis: AI-Generated Playable Minecraft (2024)

Oasis is the world's first playable AI-generated game, reimagining Minecraft entirely through
real-time AI generation. Every frame is generated by AI using next-frame prediction trained on
millions of hours of Minecraft footage. The model takes player actions as input and generates
the corresponding next frame in real-time.

Technical specifications:

- Architecture: Next-Frame Prediction DiT + Action Conditioning
- Latent channels: 16
- Default: 1 frames at 30 FPS
- Supports I2V: Yes | T2V: Yes | V2V: No

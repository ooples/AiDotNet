`Options & Configuration` · `AiDotNet.Diffusion.Audio`

Configuration for spectrogram generation.

## Properties

| Property | Summary |
|:-----|:--------|
| `FFTSize` | FFT window size. |
| `HopLength` | Hop length between frames. |
| `MaxFrequency` | Maximum frequency in Hz. |
| `MinFrequency` | Minimum frequency in Hz. |
| `NumMelBins` | Number of mel frequency bins. |
| `SampleRate` | Audio sample rate in Hz. |
| `UseLogScale` | Whether to use log-scale magnitude. |

`Options & Configuration` · `AiDotNet.Diffusion.Video`

Configuration for AnimateDiff motion modules.

## Properties

| Property | Summary |
|:-----|:--------|
| `PreserveFirstFrame` | Gets or sets whether to preserve the first frame in image-to-video. |
| `TemporalFalloff` | Gets or sets the temporal falloff for edge frames. |
| `TemporalSmoothing` | Gets or sets the temporal smoothing strength (0-1). |
| `Version` | Gets or sets the motion module version. |

## Methods

| Method | Summary |
|:-----|:--------|
| `Clone` | Clones this configuration. |

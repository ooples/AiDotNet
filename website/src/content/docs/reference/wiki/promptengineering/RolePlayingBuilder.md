`Helpers & Utilities` · `AiDotNet.PromptEngineering.Templates`

Builder for constructing role-playing templates fluently.

## Methods

| Method | Summary |
|:-----|:--------|
| `AsRole(String)` | Sets the role or persona. |
| `Build` | Builds the role-playing template. |
| `WithAttribute(String,String)` | Adds a custom attribute to the persona. |
| `WithConstraints(String)` | Sets constraints for the role. |
| `WithExpertise(String)` | Adds an area of expertise. |
| `WithExpertise(String[])` | Adds multiple areas of expertise. |
| `WithPersonality(String)` | Sets the personality and communication style. |
| `WithTask(String)` | Sets the task to perform. |

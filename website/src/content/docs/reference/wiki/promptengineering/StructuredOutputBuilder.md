`Helpers & Utilities` · `AiDotNet.PromptEngineering.Templates`

Builder for constructing structured output templates fluently.

## Methods

| Method | Summary |
|:-----|:--------|
| `AddField(String,String)` | Adds a field to the schema (for JSON/YAML formats). |
| `Build` | Builds the structured output template. |
| `WithExample(String)` | Sets an example of expected output. |
| `WithFormat(OutputFormat)` | Sets the output format. |
| `WithSchema(String)` | Sets the schema definition. |
| `WithTask(String)` | Sets the task description. |

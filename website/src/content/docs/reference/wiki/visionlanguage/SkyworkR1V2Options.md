`Options & Configuration` · `AiDotNet.VisionLanguage.Reasoning`

Configuration options for Skywork R1V2: hybrid RL (MPO + GRPO) for multimodal reasoning SOTA.

## For Beginners

These options configure the SkyworkR1V2 model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `SkyworkR1V2Options(SkyworkR1V2Options)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `EnableHybridRL` | Gets or sets whether to use hybrid RL alignment (MPO + GRPO). |

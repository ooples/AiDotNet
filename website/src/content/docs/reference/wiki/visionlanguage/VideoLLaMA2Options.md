`Options & Configuration` · `AiDotNet.VisionLanguage.VideoLanguage`

Configuration options for VideoLLaMA 2: spatial-temporal convolution for video tokens.

## For Beginners

These options configure the VideoLLaMA2 model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `VideoLLaMA2Options(VideoLLaMA2Options)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `EnableSpatialTemporalConv` | Gets or sets whether to use spatial-temporal convolution for video token compression. |

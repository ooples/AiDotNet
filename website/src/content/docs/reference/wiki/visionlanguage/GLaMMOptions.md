`Options & Configuration` · `AiDotNet.VisionLanguage.Grounding`

Configuration options for GLaMM: pixel-level grounded LMM generating text and segmentation masks.

## For Beginners

These options configure the GLaMM model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `GLaMMOptions(GLaMMOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `EnablePixelGrounding` | Gets or sets whether to produce pixel-level grounding masks. |
| `MaskDim` | Gets or sets the segmentation mask feature dimension. |

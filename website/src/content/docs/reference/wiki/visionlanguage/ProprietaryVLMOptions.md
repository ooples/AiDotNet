`Options & Configuration` · `AiDotNet.VisionLanguage.Proprietary`

Base configuration options for proprietary VLM reference implementations.

## For Beginners

These options configure the Proprietary model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `ProprietaryVLMOptions` | Initializes a new instance with default values. |
| `ProprietaryVLMOptions(ProprietaryVLMOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `LanguageModelName` | Gets or sets the language model backbone name. |
| `MaxContextLength` | Gets or sets the maximum context window length in tokens. |
| `Provider` | Gets or sets the proprietary model provider name. |

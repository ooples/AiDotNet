`Options & Configuration` · `AiDotNet.VisionLanguage.Medical`

Base configuration options for medical domain vision-language models.

## For Beginners

These options configure the Medical model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `MedicalVLMOptions` | Initializes a new instance with default values. |
| `MedicalVLMOptions(MedicalVLMOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `LanguageModelName` | Gets or sets the language model backbone name. |
| `MaxOutputTokens` | Gets or sets the maximum output token length for report generation. |
| `MedicalDomain` | Gets or sets the medical domain specialization. |

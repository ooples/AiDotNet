`Options & Configuration` · `AiDotNet.VisionLanguage.Reasoning`

Configuration options for Kimi-VL-Thinking: long chain-of-thought reasoning with RL alignment.

## For Beginners

These options configure the KimiVLThinking model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `KimiVLThinkingOptions(KimiVLThinkingOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `ActiveParameters` | Gets or sets the active parameter count in billions. |
| `EnableLongThinking` | Gets or sets whether to enable extended thinking chains. |
| `TotalParameters` | Gets or sets the total parameter count in billions. |

`Options & Configuration` · `AiDotNet.VisionLanguage.Generative`

Options for BLIP-3 (xGen-MM) with interleaved data and any-to-any generation.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `BLIP3Options(BLIP3Options)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `NumQFormerHeads` | Gets or sets the number of Q-Former attention heads. |
| `NumQFormerLayers` | Gets or sets the number of Q-Former layers. |
| `NumQueryTokens` | Gets or sets the number of learnable query tokens. |
| `QFormerDim` | Gets or sets the Q-Former hidden dimension. |
| `UseInterleavedData` | Gets or sets whether interleaved image-text training is enabled. |

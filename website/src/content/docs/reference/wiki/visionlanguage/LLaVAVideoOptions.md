`Options & Configuration` · `AiDotNet.VisionLanguage.VideoLanguage`

Configuration options for LLaVA-Video: synthetic dataset-trained video instruction model.

## For Beginners

These options configure the LLaVAVideo model. Default values follow the original paper settings.

## Constructors

| Constructor | Summary |
|:-----|:--------|
| `LLaVAVideoOptions(LLaVAVideoOptions)` | Initializes a new instance by copying from another instance. |

## Properties

| Property | Summary |
|:-----|:--------|
| `ImageChannels` | Number of channels per video frame. |
| `PatchSize` | Patch size for the vision encoder. |

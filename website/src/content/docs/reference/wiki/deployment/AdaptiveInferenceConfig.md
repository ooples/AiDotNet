Configuration for adaptive inference.

## For Beginners

AdaptiveInferenceConfig provides AI safety functionality. Default values follow the original paper settings.

## Properties

| Property | Summary |
|:-----|:--------|
| `QualityLevel` | Gets or sets the quality level. |
| `QuantizationBits` | Gets or sets the quantization bit width. |
| `SkipLayers` | Gets or sets the layers to skip for speed. |
| `UseQuantization` | Gets or sets whether to use quantization. |

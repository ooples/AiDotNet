`Models & Types` · `AiDotNet.Deployment.Export.Onnx`

Represents an ONNX computational graph.

## For Beginners

OnnxGraph provides AI safety functionality. Default values follow the original paper settings.

## Properties

| Property | Summary |
|:-----|:--------|
| `DocString` | Gets or sets the graph documentation string. |
| `Initializers` | Gets the initializers (weights and biases). |
| `Inputs` | Gets the list of input nodes. |
| `Name` | Gets or sets the name of the graph. |
| `Operations` | Gets the list of operations in the graph. |
| `OpsetVersion` | Gets or sets the ONNX opset version. |
| `Outputs` | Gets the list of output nodes. |

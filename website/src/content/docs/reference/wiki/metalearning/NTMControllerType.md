Controller type for Neural Turing Machine.

## For Beginners

The controller is the "brain" of the NTM that
decides what to read from and write to memory. LSTM is better for
sequential data, while MLP works for fixed-size inputs.

## Fields

| Field | Summary |
|:-----|:--------|
| `LSTM` | LSTM controller for sequential data processing. |
| `MLP` | MLP controller for fixed-size data processing. |

`Options & Configuration` · `AiDotNet.MetaLearning.Options`

Configuration options for HyperMAML (hypernetwork-based MAML initialization).

## For Beginners

HyperMAML improves MAML's starting point:

- MAML: Same initialization for every task
- HyperMAML: Custom initialization for each task, generated by a hypernetwork

The hypernetwork looks at the support set and produces an initialization that's
already close to the task-specific optimum, requiring fewer adaptation steps.

## How It Works

HyperMAML uses a hypernetwork to generate task-specific initial parameters for MAML,
rather than using a single shared initialization. The hypernetwork conditions on
support set statistics to produce a better starting point for each task.

Memory initialization strategies for NTM.

## For Beginners

How to fill the memory when starting.
Zeros is simplest, Random adds some noise, Learned uses trained patterns.

## Fields

| Field | Summary |
|:-----|:--------|
| `Learned` | Initialize with learned patterns. |
| `Random` | Initialize with small random values. |
| `Zeros` | Initialize all memory values to zero. |

`Models & Types` · `AiDotNet.MetaLearning.Algorithms`

Adapted model for Neural Process family algorithms.

## For Beginners

This is the output of a Neural Process after it has seen
some context examples. It stores the aggregated representation of what the model learned
from those examples and uses it to make predictions at new target points. Unlike
gradient-based approaches, adaptation happens in a single forward pass.

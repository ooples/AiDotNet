`Enums` · `AiDotNet.Models.Options`

Flags for selecting which technical indicators to calculate.

## For Beginners

Technical indicators are mathematical tools that analyze price patterns.
Traders use these to identify trends, momentum, overbought/oversold conditions, and potential
reversal points.

## Fields

| Field | Summary |
|:-----|:--------|
| `ADX` | Average Directional Index (ADX) - Trend strength indicator (0-100). |
| `ATR` | Average True Range (ATR) - Volatility indicator. |
| `All` | All available technical indicators. |
| `BollingerBands` | Bollinger Bands - Volatility bands around a moving average. |
| `CCI` | Commodity Channel Index (CCI) - Momentum indicator. |
| `DEMA` | Double Exponential Moving Average (DEMA) - Reduces lag of standard EMA. |
| `EMA` | Exponential Moving Average (EMA) - Weighted average giving more weight to recent prices. |
| `MACD` | Moving Average Convergence Divergence (MACD) - Trend-following momentum indicator. |
| `Momentum` | All momentum indicators. |
| `MovingAverages` | All moving average indicators. |
| `None` | No indicators. |
| `OBV` | On-Balance Volume (OBV) - Volume-based momentum indicator. |
| `RSI` | Relative Strength Index (RSI) - Momentum oscillator (0-100). |
| `SMA` | Simple Moving Average (SMA) - Equal-weighted average of past N prices. |
| `StochasticOscillator` | Stochastic Oscillator - Momentum indicator (0-100). |
| `TEMA` | Triple Exponential Moving Average (TEMA) - Further reduces lag using three EMAs. |
| `Volatility` | All volatility indicators. |
| `Volume` | All volume-based indicators. |
| `WMA` | Weighted Moving Average (WMA) - Linear-weighted average of past N prices. |
| `WilliamsR` | Williams %R - Momentum indicator (-100 to 0). |

`Models & Types` · `AiDotNet.Models`

Lineage information for a model showing its origin and dependencies.

## Properties

| Property | Summary |
|:-----|:--------|
| `CreatedAt` | Gets or sets the creation timestamp. |
| `Creator` | Gets or sets the creator/user who trained this model. |
| `ExperimentId` | Gets or sets the experiment ID that produced this model. |
| `ModelName` | Gets or sets the model name. |
| `ParentModel` | Gets or sets parent model information (if this was derived from another model). |
| `ParentVersion` | Gets or sets the parent model version. |
| `RunId` | Gets or sets the run ID that produced this model. |
| `TrainingDataSource` | Gets or sets the training data source. |
| `Version` | Gets or sets the version. |

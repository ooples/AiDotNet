`Models & Types` · `AiDotNet.Models`

Contains results from red teaming adversarial testing.

## Properties

| Property | Summary |
|:-----|:--------|
| `AdversarialPrompts` | Gets or sets the adversarial prompts that were tested. |
| `AverageSeverity` | Gets or sets the average severity of successful attacks. |
| `ModelResponses` | Gets or sets the model's responses to adversarial prompts. |
| `SeverityScores` | Gets or sets the severity scores for each vulnerability (0-1). |
| `SuccessRate` | Gets or sets the overall red teaming success rate. |
| `SuccessfulAttacks` | Gets or sets which prompts successfully caused misaligned behavior. |
| `Vulnerabilities` | Gets or sets detailed descriptions of vulnerabilities found. |
| `VulnerabilityTypes` | Gets or sets the types of vulnerabilities found. |

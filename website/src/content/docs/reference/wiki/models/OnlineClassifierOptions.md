`Options & Configuration` · `AiDotNet.Models.Options`

Base configuration options for online (incremental) classifiers.

## For Beginners

These options control how online classifiers learn
from streaming data and adapt to changes over time.

## Properties

| Property | Summary |
|:-----|:--------|
| `InitialNumClasses` | Gets or sets the initial number of classes. |
| `RandomSeed` | Gets or sets the random seed for reproducible results. |

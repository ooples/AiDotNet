`Models & Types` · `AiDotNet.Models`

Contains statistics about training speed and progress.

## How It Works

**For Beginners:** This tracks how fast your model is training and estimates
how long until it's done.

## Properties

| Property | Summary |
|:-----|:--------|
| `ElapsedTime` | Gets or sets the total elapsed time. |
| `EstimatedTimeRemaining` | Gets or sets the estimated time remaining. |
| `IterationsCompleted` | Gets or sets the number of iterations completed. |
| `IterationsPerSecond` | Gets or sets the average iterations per second. |
| `ProgressPercentage` | Gets or sets the current progress percentage (0-100). |
| `SecondsPerIteration` | Gets or sets the average time per iteration in seconds. |
| `TotalIterations` | Gets or sets the total number of iterations planned. |

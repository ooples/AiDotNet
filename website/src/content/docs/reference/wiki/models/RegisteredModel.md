`Models & Types` · `AiDotNet.Models`

Represents a registered model in the model registry with its metadata and versioning information.

## Properties

| Property | Summary |
|:-----|:--------|
| `CreatedAt` | Gets or sets when this model was registered. |
| `Description` | Gets or sets the description of this model version. |
| `LastModifiedAt` | Gets or sets when this model was last modified. |
| `Metadata` | Gets or sets the model metadata. |
| `ModelCard` | Gets or sets the Model Card for this model version. |
| `ModelId` | Gets or sets the unique identifier for this model. |
| `Name` | Gets or sets the name of the model. |
| `Stage` | Gets or sets the current stage of the model. |
| `StoragePath` | Gets or sets the storage path for the model. |
| `Tags` | Gets or sets the tags associated with this model. |
| `Version` | Gets or sets the version number. |

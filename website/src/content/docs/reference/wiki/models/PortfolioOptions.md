`Options & Configuration` · `AiDotNet.Models.Options`

Configuration options for portfolio optimization models.

## Properties

| Property | Summary |
|:-----|:--------|
| `AllowShortSelling` | Whether to allow short selling (negative weights). |
| `MaxWeight` | Maximum weight for a single asset (concentration limit). |
| `NumAssets` | The number of assets in the portfolio. |

## Methods

| Method | Summary |
|:-----|:--------|
| `Validate` | Validates the portfolio options. |

`Options & Configuration` · `AiDotNet.Models.Options`

Global early stopping configuration that spans multiple training stages.

## Properties

| Property | Summary |
|:-----|:--------|
| `MaxTotalTime` | Gets or sets the maximum total training time across all stages. |
| `StopOnStageFailure` | Gets or sets whether to stop the entire pipeline if any stage fails. |
| `TargetMetricValue` | Gets or sets the target metric value to achieve (stops when reached). |

`Options & Configuration` · `AiDotNet.Models.Options`

Configuration options for the NeuralStressTest model.

## For Beginners

Stress testing simulates extreme market scenarios.
These options control how large the model is and how many scenarios it
tries to generate or evaluate.

## Properties

| Property | Summary |
|:-----|:--------|
| `DropoutRate` | Dropout rate. |
| `HiddenDimension` | Hidden dimension size. |
| `NumScenarios` | Number of scenarios to generate/evaluate. |

`Options & Configuration` · `AiDotNet.Models.Options`

Base options for risk models.

## Properties

| Property | Summary |
|:-----|:--------|
| `ConfidenceLevel` | Confidence level for risk metrics (e.g., 0.95, 0.99). |
| `LossFunction` | Optional loss function override for training. |
| `NumFeatures` | Number of input features used for risk calculation. |
| `TimeHorizon` | Time horizon for risk estimation in days. |

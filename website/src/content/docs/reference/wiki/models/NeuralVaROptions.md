`Options & Configuration` · `AiDotNet.Models.Options`

Configuration options for the NeuralVaR model.

## For Beginners

These options set the size and configuration of the neural
network that predicts Value-at-Risk (the worst-case loss estimate).

## Properties

| Property | Summary |
|:-----|:--------|
| `HiddenDimension` | Size of hidden layers. |
| `HiddenLayers` | Number of hidden layers in the network. |

## Methods

| Method | Summary |
|:-----|:--------|
| `Validate` | Validates the NeuralVaR options. |

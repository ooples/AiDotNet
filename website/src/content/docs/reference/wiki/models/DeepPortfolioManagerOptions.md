`Options & Configuration` · `AiDotNet.Models.Options`

Configuration options for the DeepPortfolioManager model.

## For Beginners

These options control the deep portfolio manager,
which uses neural networks to directly predict optimal portfolio weights.

## Properties

| Property | Summary |
|:-----|:--------|
| `AllowShortSelling` | Whether to allow short selling (negative weights). |
| `MaxWeight` | Maximum weight for a single asset (concentration limit). |

## Methods

| Method | Summary |
|:-----|:--------|
| `Validate` | Validates the options. |

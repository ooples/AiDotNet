`Options & Configuration` · `AiDotNet.Models.Options`

Configuration options for the Financial A2C (Advantage Actor-Critic) trading agent.

## For Beginners

A2C is a fast actor-critic RL algorithm that trains
synchronously. These options extend the base trading agent options with
A2C-specific parameters.

## Properties

| Property | Summary |
|:-----|:--------|
| `NSteps` | Number of steps between updates. |
| `NumEnvironments` | Number of parallel environments for training. |

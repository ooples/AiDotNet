Types of sub-models available for modeling distribution parameters.

## Fields

| Field | Summary |
|:-----|:--------|
| `Additive` | Additive model using smoothing splines. |
| `Constant` | Parameter is a constant (not modeled as function of features). |
| `GradientBoosting` | Gradient boosting for flexible non-linear modeling. |
| `Linear` | Linear model: parameter = β₀ + β₁x₁ + β₂x₂ + ... |

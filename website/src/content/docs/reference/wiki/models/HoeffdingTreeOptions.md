`Options & Configuration` · `AiDotNet.Models.Options`

Configuration options for Hoeffding Tree classifier.

## Properties

| Property | Summary |
|:-----|:--------|
| `Delta` | Gets or sets the confidence level for the Hoeffding bound. |
| `GracePeriod` | Gets or sets the minimum number of samples before considering a split. |
| `MaxDepth` | Gets or sets the maximum depth of the tree. |
| `NumBins` | Gets or sets the number of bins for numeric attribute discretization. |
| `TieThreshold` | Gets or sets the tie threshold for split decisions. |

`Enums` · `AiDotNet.Finance.Enums`

Defines the different types of NLP tasks for financial text processing.

## For Beginners

Financial NLP models can perform various tasks on text data.
This enum specifies which task a model should be configured for.

## Fields

| Field | Summary |
|:-----|:--------|
| `Classification` | Text classification task (e.g., sentiment analysis, document categorization). |
| `NamedEntityRecognition` | Named Entity Recognition (NER) task. |
| `QuestionAnswering` | Question Answering (QA) task. |
| `RelationshipExtraction` | Relationship extraction task. |
| `SequenceToSequence` | Sequence-to-sequence generation task. |
| `Summarization` | Text summarization task. |
| `TokenClassification` | Token-level classification (e.g., POS tagging). |

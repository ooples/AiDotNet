| [`BenchmarkMetricValue`](/docs/reference/wiki/benchmarking/benchmarkmetricvalue/) | Represents a single metric value in a benchmark report. |
| [`BenchmarkReport`](/docs/reference/wiki/benchmarking/benchmarkreport/) | Represents a structured report from running one or more benchmark suites. |
| [`BenchmarkSuiteDescriptor`](/docs/reference/wiki/benchmarking/benchmarksuitedescriptor/) | Describes a benchmark suite and how it is categorized within AiDotNet. |

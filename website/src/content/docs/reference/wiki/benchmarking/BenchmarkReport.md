## Properties

| Property | Summary |
|:-----|:--------|
| `EndedUtc` | Gets the UTC time when the benchmark run ended. |
| `OverallStatus` | Gets the overall status for the run. |
| `StartedUtc` | Gets the UTC time when the benchmark run started. |
| `Suites` | Gets the per-suite reports for this run. |
| `TotalDuration` | Gets the total run duration. |

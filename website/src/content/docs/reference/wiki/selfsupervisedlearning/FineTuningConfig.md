`Options & Configuration` · `AiDotNet.SelfSupervisedLearning`

Configuration for fine-tuning SSL pretrained encoders.

## Properties

| Property | Summary |
|:-----|:--------|
| `BatchSize` | Batch size. |
| `EncoderLRMultiplier` | Learning rate multiplier for encoder (default: 0.1). |
| `Epochs` | Number of epochs. |
| `LearningRate` | Base learning rate. |
| `Strategy` | Fine-tuning strategy. |

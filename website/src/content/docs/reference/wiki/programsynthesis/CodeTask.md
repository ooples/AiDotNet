Defines the different types of code-related tasks that can be performed.

## For Beginners

These are different things you might want to do with code.

Just like you can do different things with text (read, write, translate, summarize),
you can do different things with code. This enum lists all the code-related tasks
the system can help with.

## How It Works

This enumeration categorizes the various operations that can be performed on code,
from understanding and generation to transformation and quality assurance.

## Fields

| Field | Summary |
|:-----|:--------|
| `BugDetection` | Bug detection task - identifying potential errors and issues in code. |
| `BugFixing` | Bug fixing task - automatically repairing identified bugs in code. |
| `CloneDetection` | Clone detection task - identifying duplicate or similar code. |
| `CodeReview` | Code review task - analyzing code quality and suggesting improvements. |
| `Completion` | Code completion task - suggesting how to complete partial code. |
| `Documentation` | Code documentation task - generating documentation for code. |
| `Generation` | Code generation task - creating new code from specifications or descriptions. |
| `Refactoring` | Code refactoring task - improving code structure without changing functionality. |
| `Search` | Code search task - finding relevant code based on queries. |
| `Summarization` | Code summarization task - generating natural language descriptions of code. |
| `TestGeneration` | Test generation task - automatically creating test cases for code. |
| `Translation` | Code translation task - converting code from one language to another. |
| `Understanding` | Code understanding task - analyzing and comprehending code semantics. |
